# Test Prep Unit Plan — Intensive Reading | Quarter 3-4 Framework

## Purpose

This framework provides the complete structure for building test prep units in Quarters 3-4 of 10th grade Intensive Reading. Each unit uses 4 texts (2 fiction, 2 nonfiction) across 16 instructional days, with the Attack the Passage protocol as the core strategy.

**The shift from Q1-Q2 to Q3-Q4:** In Quarters 1-2, units focus on deep analysis of single texts with benchmark-specific organizers and gradual release. In Quarters 3-4, units shift to test-taking stamina, cold reads, and evidence-based elimination. The Attack the Passage protocol replaces the graphic organizer as the primary student tool.

---

## Unit Framework Overview

| Component | Details |
|-----------|---------|
| **Texts Per Unit** | 4 texts (2 fiction, 2 nonfiction) |
| **Days Per Text** | 4 days |
| **Total Unit Length** | 16 instructional days |
| **Class Period** | 90 minutes |
| **Rotation Model** | 2 groups rotating through teacher-led and independent centers |
| **Questions Per Text** | 10 multiple choice |
| **Core Strategy** | Attack the Passage Protocol (C-U-B + STOP) |
| **Writing Model** | CER (literary) or RACE (informational) — one per text based on genre |
| **Tracking** | Per-text tracker + unit progress tracker + error analysis |

---

## Unit Goals

1. Build authentic test-taking stamina through cold reads
2. Develop metacognitive awareness of question types and answer traps
3. Practice evidence-based justification using CER/RACE model
4. Track progress and identify error patterns
5. Apply STOP strategy consistently for answer elimination

| Benchmark | Focus |

| ELA.10.R.1.2 | Theme (universal themes, development) |

### Nonfiction Texts (2 per unit)

| Benchmark | Focus |
|-----------|-------|
| ELA.10.R.2.1 | Text Structure and Features |
| ELA.10.R.2.2 | Central Idea |
| ELA.10.R.2.3 | Purpose and Perspective |
| ELA.10.R.2.4 | Argument (claims, evidence, reasoning) |
| ELA.10.R.3.4 | Rhetoric (rhetorical devices, appeals) |

**Text selection guidance:** Each fiction text should target 2-3 literary benchmarks. Each nonfiction text should target 2-3 informational benchmarks. The 10 MC questions per text should assess the targeted benchmarks with an 80% benchmark / 20% vocabulary split (8 benchmark questions, 2 vocabulary questions).

## The 4-Day Cycle Per Text

### Day 1: Cold Read + Answer

| Time | Whole Class | Teacher-Led Center (45 min) | Independent Center (45 min) |
|------|------------|---------------------------|---------------------------|
| 0-5 min | Bellringer (vocab or review) | — | — |
| 5-50 min | — | Cold read + 10 MC + quick justifications | Phase 1: Attack the Questions (preview questions for upcoming cold read) |
| 50-90 min | — | Groups swap | Groups swap |

**Teacher-Led Focus:**
- Monitor cold read under test conditions (30 min read time)
- Students answer 10 MC questions
- Students write quick 1-sentence justifications for each answer
- Ticket checkpoint: Complete = ticket earned

**Independent Focus:**
- Students receive questions ONLY (no passage yet)
- Apply Phase 1: Circle-Underline-Box the questions
- Identify what each question is asking
- Predict what evidence they'll need
- Write "hunt predictions" (3-5 words per question)
- **This prepares them for their cold read when they rotate**

**Key design:** The independent center on Day 1 gives students who haven't done the cold read yet a PURPOSE-DRIVEN preview. When they rotate to teacher-led, they already know what they're hunting for.

---

### Day 2: Question Stem Analysis

| Time | Whole Class | Teacher-Led Center (45 min) | Independent Center (45 min) |
|------|------------|---------------------------|---------------------------|
| 0-5 min | Bellringer (vocab in context) | — | — |
| 5-50 min | — | Model question stem analysis, identify benchmarks | Question stem worksheet |
| 50-90 min | — | Groups swap | Groups swap |

**Teacher-Led Focus:**
- Deep dive into what each question is really asking
- Identify the benchmark being assessed for each question
- Connect question types to reading skills
- Model how to C-U-B effectively with think-alouds
- Focus on Tricky vs. Proven distinction (the #1 error students make)

**Independent Focus:**
- Worksheet: Categorize question types (literal, inference, analysis, evaluation)
- Match questions to benchmarks
- Identify key words in stems
- Write "hunt predictions" for each question
- Return to annotated passage and write Q# in margins next to evidence

---

### Day 3: Answer Choice Elimination + CER/RACE

| Time | Whole Class | Teacher-Led Center (45 min) | Independent Center (45 min) |
|------|------------|---------------------------|---------------------------|
| 0-5 min | Bellringer (vocab application) | — | — |
| 5-50 min | — | Model STOP strategy, guided elimination practice | Write CER/RACE justifications for each answer |
| 50-90 min | — | Groups swap | Groups swap |

**Teacher-Led Focus:**
- Model STOP strategy on 2-3 questions (full S/T/O/P labeling with justifications)
- Students practice elimination on remaining questions
- Discuss text features that help locate evidence
- Focus on: "You said B is Tricky — tell me WHY it's tricky and not Proven"
- Individual conferencing on error patterns

**Independent Focus:**
- Write full CER (literary) or RACE (informational) justification for each MC answer
- Use provided sentence frames
- Complete tracker columns: Benchmark, Answer, Justification
- Self-check: Does my justification point to specific text evidence?

---

### Day 4: Review Game + Extended CER/RACE

| Time | Activity | Grouping |
|------|---------|----------|
| 0-5 min | Bellringer (vocab quiz) | Whole class |
| 5-25 min | Review game (answers revealed) | Whole class |
| 25-60 min | Rotation 1: Guided extended CER/RACE / Independent: Draft extended response | Split |
| 60-90 min | Rotation 2: Groups swap | Split |
| Final 5 min | Complete tracker, self-assessment | Whole class |

**Review Game Options (20 min):** See `attack-the-passage` skill for full game descriptions.
- 4 Corners
- Defense Court
- Beat the Teacher
- Question Stem Jeopardy
- Error Auction

**Teacher-Led Focus:**
- Guide extended CER/RACE writing
- Individual conferencing on error patterns
- Model how to expand quick justification into full CER/RACE paragraph

**Independent Focus:**
- Write extended CER/RACE response for 1-2 selected questions
- Complete tracker with corrections and error type codes
- Self-assessment checklist

## CER / RACE Justification Model

### Quick Justification (Day 1 — during cold read)

**Format:** One sentence
"The answer is ___ because [brief reason connected to text]."

### Extended Justification (Days 3-4 — during review)

**Use CER for literary texts:**

| Component | Purpose | Sentence Starters |
|-----------|---------|-------------------|
| **CLAIM** | State your answer and what it means | "The correct answer is ___ because..." |
| **EVIDENCE** | Quote from text with paragraph number | "In paragraph ___, the text states '___.' " |
| **REASONING** | Explain how evidence proves your claim | "This proves... because..." |

**Use RACE for informational texts:**

| Component | Purpose | Sentence Starters |
|-----------|---------|-------------------|
| **RESTATE** | Turn the question into a statement | "The question asks about..." |
| **ANSWER** | Provide a direct, specific answer | "The author..." / "The central idea is..." |
| **CITE** | Quote text evidence with paragraph # | "In paragraph ___, the text states '___.' " |
| **EXPLAIN** | Explain how evidence proves the answer | "This shows... because..." |

### CER Sentence Frames (for student materials)

```
Claim:
- The correct answer is ___ because it shows that _____________.
- Answer ___ is correct because the author demonstrates _____________.

Evidence:
- In paragraph ___, the text states, "___________."
- The author writes, "_______________" (para ___).
- According to the passage, "_______________" (para ___).

Reasoning:
- This evidence proves the answer because _____________.
- This shows that ___ is correct because _____________.
- The quote supports this answer because _____________.
```

## Differentiation

### For Struggling Readers
- Provide questions before reading (always do Phase 1 first)
- Reduce to 5 MC questions if needed
- Offer paragraph-specific hints ("Check para 3-4")
- Allow use of Attack the Passage reference card during cold read
- Pull for small group during Day 3 teacher-led center for STOP reteach
- Modified tracker with pre-filled benchmark column

### For Advanced Readers
- Add "explain why wrong answers are wrong" to tracker (full S/T/O labeling)
- Write CER/RACE for multiple questions, not just selected ones
- Create their own MC questions for peers (with intentional S/T/O/P distractor design)
- Lead review game discussions
- Extend to Part A / Part B question format

## Success Criteria

### Student Success Looks Like:
- [ ] Completing cold read and all MC within time limit
- [ ] Using C-U-B consistently on questions AND passage
- [ ] Applying STOP to eliminate before selecting
- [ ] Writing evidence-based justifications (not guessing)
- [ ] Identifying personal error patterns on tracker
- [ ] Showing growth from cold read score to corrected score across 4 texts

### Teacher Success Looks Like:
- [ ] Students engaged during cold read (not rushing/guessing)
- [ ] Visible annotations on student papers (C-U-B marks, Q# in margins)
- [ ] Students can articulate WHY an answer is right or wrong
- [ ] Error type data shows addressable patterns
- [ ] Review games generate discussion, not just answer-checking
- [ ] Growth trend visible on unit progress tracker
