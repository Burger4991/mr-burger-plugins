# Review & Assessment Prep Activities

Engaging review protocols for revision, peer feedback, and test preparation.

## Protocol Selection

| Protocol | Time | Best For | Day |
|----------|:----:|----------|:---:|
| Speed Dating Review | 15-20 min | Revision, peer feedback | Days 4-5 |
| Kahoot/Quizizz | 10-15 min | Formative assessment, competitive review | Days 5-6 |

---

## SPEED DATING PEER REVIEW

**Time:** 15-20 minutes | **Grouping:** Rotating pairs

### Setup (2 min)
- Desks in two rows facing each other
- Row A stays seated
- Row B rotates
- Each student has draft (organizer, CR, etc.)

### Round Structure (3-4 min each)

**Partner A (1-2 min):**
- Reads work aloud to Partner B
- Partner B listens (no interrupting)

**Partner B (1-2 min):**
- Gives feedback using frames:
  - "One strength is..."
  - "One area to improve is..."
  - "Have you considered...?"

**Partner A:**
- Takes notes on feedback
- Asks clarifying questions

**Rotation:** Row B shifts one seat, Row A stays

### Complete 3-4 Rounds
- New partner each round
- 3-4 minutes per round
- Students receive feedback from 4 different peers

### Feedback Structures

**Two Stars and a Wish:**
- ⭐ "One thing you did well..."
- ⭐ "Another strength..."
- 🌟 "One way to make this better..."

**Glow and Grow:**
- 🟢 Glow: "What's working well..."
- 🟡 Grow: "What could improve..."

**TAG Feedback:**
- T = Tell what you like
- A = Ask a question
- G = Give a suggestion

### Final Reflection (3 min)
- Review all feedback received
- Identify patterns
- Choose ONE thing to revise immediately
- Make revision (3 min)

### Sentence Frames
- "Your evidence from ¶___ is strong because..."
- "You could strengthen this by adding..."
- "I'm confused about... Can you clarify?"
- "Your reasoning would be clearer if..."

### Scaffolds

**Struggling:** Feedback frames on cards, model first, allow reading from notes
**Advanced:** Content-specific feedback, reference rubric, explain WHY

### Materials
- Student drafts
- Feedback frames (printed at each seat)
- Rubric reference
- Visible timer
- Revision tools (pens, highlighters)

## KAHOOT/QUIZIZZ COMPETITIONS

**Time:** 10-15 minutes | **Grouping:** Individual or teams

### Pre-Class Setup
- Create game with 10-15 MC questions
- Range: DOK 1 (recall) → DOK 3 (analysis)
- Include: Comprehension, vocabulary, benchmark skills, test strategies

### Launch (1 min)
- Project game code
- Students join on devices
- Low-tech option: Whiteboards + projected questions

### Gameplay (10-12 min)

**Individual Mode:**
- Students answer independently
- Timed questions (20-30 sec each)
- Leaderboard shows top 5
- Points for speed + accuracy

**Team Mode:**
- Partners/groups discuss before answering
- Must reach consensus
- Reduces test anxiety
- Builds collaboration

**After Each Question:**
- Show correct answer
- Discuss: "Why is B the best answer?"
- Address misconceptions
- Review strategies:
  - "Eliminate obviously wrong first"
  - "Look for key words"
  - "Go back to text"

### Debrief (2-3 min)
- Announce top scores (celebrate!)
- "What question types were hardest?"
- "What strategies helped?"
- "What to review before test?"

**Use Data:**
- Which questions did most miss?
- Reteach those concepts
- Target small group instruction

### Question Design Tips

**Good Questions:**
- Text-based (require going back to passage)
- Realistic (similar to actual test)
- Range of difficulty
- Clear correct answer
- Plausible distractors

**DOK Examples:**
- DOK 1: "What is the setting of this text?"
- DOK 2: "What is the author's main purpose in ¶3?"
- DOK 3: "How does repetition affect the audience?"

### Low-Tech Alternatives
- Paper quiz with hand-raised answers
- Students hold up A/B/C/D cards
- Whiteboards with answers
- Quick visual scan for understanding

### Modifications

**Struggling:** Team mode, longer time per question, fewer questions, mix easy DOK 1-2
**Advanced:** All DOK 3, explain answer choice, create questions for next game

### Materials
- Kahoot/Quizizz account (free)
- Student devices
- Projector/screen
- Game PIN code
- OR: Whiteboards, answer cards

|-----|---------------------|

## Troubleshooting

### "Only some students participate"
- Use talking chips (forces equal turns)
- Assign roles
- Start with lower-stakes Think-Pair-Share
- Build respect culture first

### "Protocols take too long"
- Teach early in year (frontload)
- Use same protocols repeatedly
- Streamline transitions
- Cut debrief if needed

### "No tech available"
- Kahoot → Paper quiz with hands raised
- Gallery Walk → Pass papers instead of walking
- Annotation Stations → Whole-class with different colors
