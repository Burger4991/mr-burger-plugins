# Multiple Choice Question Generation Protocol

## PURPOSE

This protocol ensures MC questions are:
1. **Benchmark-aligned** (use Planning Card stems)
2. **Text-based** (require going back to passage)
3. **STOP-formatted** (Silly, Tricky, Opposite, Proven choices)
4. **DOK-appropriate** (matched to skill complexity)
5. **Student-accessible** (clear, not trick questions)

## ⚠️ MANDATORY: DISTRICT QUESTION STEMS

**Before generating ANY MC question for ANY benchmark, you MUST read `references/district-planning-guide.md` and:**
1. Use ONLY the question stems listed there as your starting templates
2. Adapt stems by filling in blanks with text-specific details
3. Check Assessment Limits to ensure your item doesn't violate constraints
4. Use Achievement Level Descriptors to calibrate difficulty
5. Follow the Planning Card Analytical Processes to ensure questions align with how students are taught to analyze text

**Do NOT invent new question stem patterns.** Every MC stem should trace back to a district-provided template. The reference file contains stems, assessment limits, achievement level descriptors, and step-by-step analytical processes for all benchmarks (R.1.1–R.3.4, V.1.2, V.1.3).

## PROTOCOL 1: BELLRINGER VOCABULARY MC QUESTIONS

Bellringer vocabulary MC questions come in TWO formats. See bellringer-builder skill for complete bellringer design protocols.

### Format A: Context Clue Questions (from unit passage)

**Stem:** "What does [word] most likely mean?"
**Source:** Exact sentence from unit text with bold target word + paragraph reference
**Answer choices:** 4 options using STOP format — definitions/synonyms

**Criteria for word selection:**
- Tier 2 academic word from unit text
- Appears in sentence with strong context clues
- Students unlikely to know definition already
- NOT proper nouns, NOT Tier 1 basic words

**Context Clue Types** (sentence must contain one):
- **Definition**: Word is defined in sentence ("X, which means...")
- **Example**: Word is illustrated ("X such as Y and Z")
- **Contrast**: Word is contrasted with opposite ("Unlike X, Y is...")
- **Restatement**: Word is restated in simpler terms ("X, or in other words...")
- **Inference**: Multiple surrounding clues combined (no direct definition)

**Example (Context Clue format):**
```
"Knowing that Mrs. Mallard was afflicted with a heart trouble, great care
was taken to break to her as gently as possible the news of her husband's
death." (¶1)

What does afflicted most likely mean?
A) Cured of [O - Opposite]
B) Suffering from [P - PROVEN]
C) Ignoring [S - Silly, doesn't match context]
D) Celebrating [T - Tricky, positive emotion doesn't fit]
```

### Format B: Word Parts / Morphology Questions (standalone sentences)

**Stem:** "What is the meaning of the bold word?"
**Source:** Standalone sentences NOT tied to a specific passage — real-world contexts
**Answer choices:** Two sub-formats:

**Type 1 — Sentence completion:** Sentence ends with colon; choices complete it.
Each completion is plausible, but only one aligns with the bold word's meaning.

```
What is the meaning of the bold word?
The new software update aims to enable users to:
A) Fix bugs and glitches in the system.
B) Customize their interface with different themes.
C) Access advanced features previously unavailable.
D) Share files securely across multiple devices.
```

**Type 2 — Direct definition:** Sentence is complete; choices are single-word/short-phrase definitions.

```
What is the meaning of the bold word?
The witness's testimony seemed to contradict the evidence
presented by the prosecution, causing confusion in the courtroom.
a) Support
b) Confirm
c) Oppose
d) Clarify
```

**Key design principle for morphology items:** All target words in a set share the same prefix/suffix/root. Students should be able to use morphological breakdown as one strategy alongside context clues and STOP elimination.

### STOP Answer Choices (Both Formats)

**Choice Order**: Randomize S/T/O/P (don't always put P as choice D)

**S (Silly) - Obviously Wrong:**
- Unrelated definition, wrong part of speech, absurd meaning
- Examples: "afflicted" → "ignoring" | "contradict" → "clarify"

**T (Tricky) - Partially Correct:**
- Synonym that sounds right but doesn't fit context, misses key nuance
- Examples: "elusive" → "colorful" | "counterpart" → "opponent"

**O (Opposite) - Reverse Meaning:**
- Antonym of correct answer
- Examples: "afflicted" → "cured of" | "contradict" → "support"

**P (Proven) - Correct Answer:**
- Matches definition supported by context clues and/or morphological analysis
- Examples: "afflicted" → "suffering from" | "contradict" → "oppose"

## PROTOCOL 2: COMPREHENSION QUESTIONS (Margin Questions → MC)

### Step 1: Triage District Margin Questions
**Keep these margin questions:**
- Aligned to benchmark
- Text-based (require evidence)
- DOK 2-3 (literal comprehension or inference)

**Revise these:**
- Yes/no questions → Convert to "Why/How"
- Trivial detail questions → Elevate to inference

**Skip these:**
- Not aligned to benchmark
- Opinion-only (no text evidence)
- Already answered in text (too easy)

### Step 2: Convert to MC Format
**Original margin question:**
"Why does the character make this choice?"

**MC format:**
"Why does [character] [action] in paragraph [#]?"

A) [S - Silly - contradicts text]
B) [T - Tricky - sounds right but wrong]
C) [O - Opposite - reverse of truth]
D) [P - PROVEN - text supports]

### Step 3: Write STOP Choices

**Use text evidence to craft choices:**

**P (Proven):** Find the answer in the text
- "The character does this because the text states [evidence]"

**O (Opposite):** Flip the proven answer
- If proven answer is "character is brave," opposite is "character is cowardly"

**T (Tricky):** Use related detail that seems right
- Pull detail from nearby paragraph that's not the answer
- Use same vocabulary but wrong meaning

**S (Silly):** Create obviously wrong option
- Contradicts basic facts from text
- States something not even mentioned

### Example:
**Text:** "Mrs. Mallard retreated to her room alone, refusing her sister's comfort."

**Margin Question:** "Why does Mrs. Mallard go to her room alone?"

**MC Conversion:**
Why does Mrs. Mallard retreat to her room alone in paragraph 4?
A) She wants to celebrate her freedom privately [P - PROVEN by later paragraphs]
B) She is too sad to be around others [T - Tricky, sounds right but incomplete]
C) Her sister told her to leave [S - Silly, contradicts text]
D) She needs to write a letter [O - Not mentioned at all]

## PROTOCOL 3: BENCHMARK APPLICATION QUESTIONS

### Step 1: Use Planning Card Question Stems

**For Theme (ELA.10.R.1.2):**
- "What theme is developed through [character/event/conflict]?"
- "How does the author use [literary element] to convey theme?"
- "Trace how the theme develops from beginning to end."

**For Rhetoric (ELA.10.R.3.4):**
- "What rhetorical device is used in paragraph [#]?"
- "How does [device] affect the audience?"
- "Why did the speaker use [device] in this section?"

**For Argument (ELA.10.R.2.4):**
- "What is the author's claim?"
- "Which evidence best supports the author's argument?"
- "How does the author address the counterclaim?"

**For Author's Purpose (ELA.10.R.2.3):**
- "What is the author's purpose in [section]?"
- "How does the author reveal their perspective?"
- "What bias is evident in paragraph [#]?"

### Step 2: Match DOK Level to Question Type

**DOK 1 (Recall):** Identify, define, locate
- "What rhetorical device is used?" (just identification)

**DOK 2 (Comprehension):** Explain, summarize, interpret
- "What theme is developed?" (requires understanding)

**DOK 3 (Analysis):** Analyze, compare, evaluate
- "How does the device affect the audience?" (requires analysis)

**DOK 4 (Extended Thinking):** Synthesize, design, create
- Rarely used in MC (better for CR questions)

### Step 3: Craft Benchmark-Specific STOP Choices

**Theme Questions:**
- P: Universal theme statement supported by text
- T: Topic (not theme) - "The text is about war" instead of "War leads to loss"
- O: Theme of different text/character
- S: Random unrelated theme

**Rhetoric Questions:**
- P: Correct device + correct effect
- T: Correct device + wrong effect OR wrong device + makes sense
- O: Opposite effect (e.g., "creates sadness" when it creates hope)
- S: Device not present in text

**Argument Questions:**
- P: Author's actual claim with text support
- T: Supporting detail mistaken for main claim
- O: Opposing side's claim
- S: Unrelated claim not in text

### Example (Theme):
```
What theme is developed through the character's realization in paragraph 10?

A) Freedom can come from unexpected circumstances. [P - PROVEN by "monstrous joy"]
B) Marriage is always oppressive. [T - Tricky, too specific, not universal]
C) Death brings only sadness. [O - Opposite of what text shows]
D) New York City is a difficult place to live. [S - Silly, not in text at all]
```

## PROTOCOL 4: ASSESSMENT QUESTIONS (Day 6)

### Quantity & Distribution
**Day 6 Mini-Assessment:**
- 10-12 MC questions total
- Mix of DOK levels:
  - 3-4 DOK 1 questions (vocabulary, recall)
  - 5-6 DOK 2 questions (comprehension, inference)
  - 2-3 DOK 3 questions (analysis, benchmark application)

### Question Sequence
**Order questions strategically:**
1. Start easy (DOK 1 vocab/recall) - build confidence
2. Middle difficulty (DOK 2 comprehension)
3. End with hardest (DOK 3 benchmark application)

**Follow text order:**
- Question 1-3: Beginning of text
- Question 4-7: Middle of text
- Question 8-10: End of text
- Question 11-12: Whole-text analysis

### Assessment-Specific STOP Guidelines

**Higher Rigor:**
- Make S less obvious (students must read carefully)
- Make T more sophisticated (requires close reading to spot flaw)
- O should be plausible (not extreme opposite)

**Example (DOK 3):**
```
How does the author's use of imagery in paragraphs 8-9 contribute to the theme?

A) The imagery creates a sense of freedom and possibility, reinforcing the theme of liberation. [P - PROVEN]
B) The imagery shows the setting's beauty, suggesting themes of nature appreciation. [T - Tricky, imagery IS beautiful, but wrong theme]
C) The imagery creates a sense of confinement, suggesting oppression. [O - Opposite effect]
D) The imagery has no connection to theme. [S - Silly for assessment level]
```

## ANSWER CHOICE WRITING TEMPLATES

### Vocabulary Questions

| STOP Type | Template | Example |
|-----------|----------|---------|
| **P** | [Context-supported definition] | "Suffering from" |
| **T** | [Synonym that doesn't fit context] | "Bothered by" (too weak) |
| **O** | [Antonym] | "Cured of" |
| **S** | [Unrelated or absurd] | "Celebrating" |

### Comprehension Questions

| STOP Type | Template | Example |
|-----------|----------|---------|
| **P** | [Answer directly stated/implied in text with evidence] | "To grieve privately" (¶4 supports) |
| **T** | [Detail from nearby paragraph, wrong answer] | "To write a letter" (mentioned elsewhere) |
| **O** | [Opposite motivation/action] | "To celebrate with her sister" |
| **S** | [Not in text at all] | "To call her husband" |

### Theme Questions

| STOP Type | Template | Example |
|-----------|----------|---------|
| **P** | [Universal theme + text evidence] | "Freedom can emerge from loss" |
| **T** | [Topic not theme OR too specific] | "The story is about marriage" |
| **O** | [Contradictory theme] | "Loss only brings sadness" |
| **S** | [Unrelated universal statement] | "Hard work leads to success" |

### Rhetoric Questions

| STOP Type | Template | Example |
|-----------|----------|---------|
| **P** | [Correct device + correct effect] | "Repetition emphasizes urgency" |
| **T** | [Correct device + wrong effect] | "Repetition creates confusion" |
| **O** | [Opposite effect] | "Makes audience feel calm" (when creates tension) |
| **S** | [Device not present] | "Uses rhyme to persuade" (no rhyme in text) |

## QUALITY CHECKLIST

Before finalizing MC questions, verify:

**Alignment:**
- [ ] Question uses benchmark-appropriate stem (from Planning Cards)
- [ ] DOK level matches question purpose (bellringer = DOK 1-2, assessment = DOK 2-3)
- [ ] All questions require text evidence (not guessable without reading)

**STOP Formatting:**
- [ ] One clearly PROVEN answer with text support
- [ ] One Silly choice (obviously wrong upon reading)
- [ ] One Tricky choice (sounds right but has fatal flaw)
- [ ] One Opposite choice (reverses correct answer)
- [ ] STOP types randomized (P not always choice D)

**Student Accessibility:**
- [ ] Question stem is clear (not confusing wording)
- [ ] All choices are similar length (don't make P obviously longer)
- [ ] Vocabulary is grade-appropriate (don't test vocab in a theme question)
- [ ] No "all of the above" or "none of the above" (weak choices)

**Text-Based:**
- [ ] Answer can be proven with specific paragraph number
- [ ] T choice uses detail from text (not made up)
- [ ] O choice references text content (reversed)
- [ ] S choice may reference text but is clearly wrong

## COMMON PITFALLS TO AVOID

### Pitfall 1: "All of the Above" Choices
**Why it's bad:** Weak test design, easy to game
**Fix:** Write 4 distinct choices with 1 clear answer

### Pitfall 2: Making P Obviously Longest
**Why it's bad:** Students pick longest answer without reading
**Fix:** Make all choices similar length (1-2 lines each)

### Pitfall 3: Using Extreme Language in Wrong Answers
**Why it's bad:** "Always, never, all, none" = obviously wrong
**Fix:** Make wrong answers plausible

### Pitfall 4: Testing Two Things at Once
**Why it's bad:** "What theme and which device?" = unclear what's being tested
**Fix:** One question = one skill

### Pitfall 5: Answer Not in Text (Inferencing Gone Too Far)
**Why it's bad:** Students can't prove answer
**Fix:** Inference should be supported by text evidence

## INTEGRATION WITH UNIT BUILDING

### When Building Bellringers (Every Day):
1. Select 3 words per day (18 total across 6 days) — see bellringer-builder skill
2. Use Protocol 1 Format A (Context Clue) for passage-based words or Format B (Morphology) for word-parts sets
3. Generate 4 STOP choices per word
4. Total output: 3 MC questions per day × 6 days = 18 questions per unit (Context Clue mode)
5. For standalone Morphology sets: 4-6 MC questions per prefix/suffix/root set

### When Building Day 1-2 Independent Comprehension:
1. Review district margin questions (triage: keep/revise/skip)
2. Use Protocol 2 (Margin → MC Conversion)
3. Create 8-10 MC questions total
4. Sequence by text order (beginning → end)

### When Building Day 6 Assessment:
1. Use Protocol 3 (Benchmark Application) for DOK 3 questions
2. Use Protocol 4 (Assessment Guidelines) for mix of DOK levels
3. Create 10-12 MC questions
4. Sequence: Easy → Medium → Hard

## AI GENERATION PROMPTS

### Prompt Template for Bellringer Questions:
```
Generate 5 context clue MC questions for this text: [paste text]

For each question:
1. Select a Tier 2 academic word with strong context clues
2. Pull the full sentence containing the word
3. Create 4 answer choices using STOP format:
   - S (Silly): Obviously wrong
   - T (Tricky): Sounds right but flawed
   - O (Opposite): Reverse meaning
   - P (Proven): Correct based on context clues

Format:
"[Sentence with word underlined]" (¶#)
What does [word] mean as used in this sentence?
A) [choice]
B) [choice]
C) [choice]
D) [choice]

Randomize the order of S/T/O/P choices (don't always put P last).
```

### Prompt Template for Benchmark Questions:
```
Generate [#] MC questions for this text using [benchmark name] Planning Card stems.

Benchmark: [ELA.10.R.X.X]
Text: [paste relevant section]
Question stems to use: [paste stems from benchmark guide]

For each question:
1. Use one of the provided Planning Card stems
2. Reference specific paragraph number
3. Create 4 STOP answer choices:
   - P: Proven by text evidence (cite paragraph)
   - T: Tricky (plausible but wrong - explain why)
   - O: Opposite of correct answer
   - S: Silly (obviously incorrect)

Include DOK level for each question (1, 2, or 3).
```
