# Self-Assessment & Reflection Tools

Tools that teach students to evaluate their own work and track growth.

## Tool Overview

| Tool | When | Purpose |
|------|------|---------|
| Self-Assessment Checklist | Before submission | Quality self-evaluation |
| Weekly Reflection Form | Every Friday | Growth tracking |
| Strategy Reminder Posters | When stuck | Problem-solving reference |
| "I Used to Think..." Protocol | End of unit | Metacognitive reflection |

---

## SELF-ASSESSMENT CHECKLIST

**Format:** Checklist attached to every major assignment
**Use:** Before submitting organizers, CR, projects

### The Checklist

**CONTENT (Answering the Question):**
- [ ] Answer the ENTIRE question/prompt
- [ ] Include required text evidence
- [ ] Explain HOW evidence supports answer
- [ ] Use academic vocabulary correctly
- [ ] Stay on topic throughout

**WRITING QUALITY:**
- [ ] Complete sentences (no fragments)
- [ ] Correct spelling
- [ ] Correct punctuation
- [ ] Organized into paragraphs
- [ ] Used transition words

**EVIDENCE & ANALYSIS:**
- [ ] Every claim has text evidence
- [ ] Used quotation marks around quotes
- [ ] Cited paragraph numbers
- [ ] Explained WHY evidence matters

**EFFORT & PRIDE:**
- [ ] This is my BEST work
- [ ] I revised after reading it over
- [ ] I would be proud to share this
- [ ] I used class time wisely
- [ ] I asked for help when needed

**SELF-EVALUATION:**
- My score: _____ / 10
- What I'm most proud of: ________________
- One thing I could improve: ________________

### Implementation

**Week 1:** Model with anonymous student work, score together
**Weeks 2-4:** Students complete before submitting, teacher provides feedback on accuracy
**Week 5+:** No work accepted without attached checklist

### Benefits

**For Students:**
- Learn what quality looks like
- Catch errors before submission
- Take ownership of quality
- Develop revision skills

**For Teachers:**
- Fewer low-quality submissions
- Insight into student thinking
- Evidence of metacognition
- Faster grading

## WEEKLY REFLECTION FORM

**Format:** Short reflection (5 min every Friday)

### The Form

**WEEKLY REFLECTION - Week of _______**

This week I learned:

One thing I got better at:

One thing that's still hard:

My effort this week was:
- [ ] Low (I could have tried harder)
- [ ] Medium (I did okay)
- [ ] High (I gave my best effort)

My goal for next week:

Something my teacher should know:

### Implementation

**Every Friday (Last 5 min):**
- Music playing (calm reflection)
- Students complete independently
- Brief partner share (optional)
- Collect forms

**Teacher Response:**
- Read over weekend
- Write 1-2 sentence comment on each
- Return Monday with feedback
- Follow up on concerns

**Student Portfolio:**
- Keep reflections in folder
- Review at midpoint/end of quarter
- "Look back - how have you grown?"

### Metacognitive Power

- **Growth Mindset:** "One thing I got better at" emphasizes GROWTH
- **Goal Setting:** Weekly goals create accountability
- **Communication:** Safe space for student concerns

## STRATEGY REMINDER POSTERS

**Format:** Large posters on classroom walls
**Use:** Students reference when stuck

### Poster 1: How to Find Text Evidence

```
1. Read the question carefully - what is it asking?
2. Go back to the text - don't answer from memory
3. Find a sentence that DIRECTLY answers the question
4. Copy evidence word-for-word (use quotation marks)
5. Explain WHY this evidence answers the question
```

### Poster 2: CER Writing Strategy

```
C = Claim (Answer the question in one sentence)
E = Evidence (Quote from text that proves your claim)
R = Reasoning (Explain HOW evidence proves your claim)

Example:
Q: What is Obama's purpose?
C: Obama's purpose is to persuade Americans to support education reform.
E: He states, "Education is the key to our future."
R: This shows persuasion because he's trying to convince the audience
   that education should be a priority.
```

### Poster 3: When I'm Stuck...

```
If I'm confused by a word → Use context clues or ask
If I don't understand a paragraph → Reread it slowly
If I don't know how to start → Look at exemplar/model
If I'm stuck on a question → Check the strategy poster
If I still need help → Ask a classmate, then ask teacher
```

### Poster 4: STOP Test-Taking Strategy

```
S = STOP and read the question carefully (underline key words)
T = THINK about what the question is asking
O = OPTIONS - read all answer choices before selecting
P = PROVE your answer with evidence from the text
```

### Implementation

- Create large, color-coded posters
- Hang permanently on walls
- Reference constantly during instruction
- Teach students to use independently

## "I USED TO THINK... NOW I THINK..." PROTOCOL

**Format:** Sentence completion at end of unit
**Use:** Last day of every unit (Day 6)

### The Protocol

**Sentence Frame:**
"I used to think __________, but now I think __________ because __________."

### Examples

*About Content:*
"I used to think Obama was just giving a speech, but now I think he was carefully using rhetoric to persuade because I noticed the repetition and emotional language."

*About Skills:*
"I used to think annotation was random highlighting, but now I think it's about marking specific things like evidence because that helps me understand better."

*About Learning:*
"I used to think I wasn't good at reading, but now I think I can improve with practice because my scores went up when I used the strategies."

### Implementation

**Last Day of Unit (5 min):**
- Project frame on board
- Students complete in notebook
- Share with partner/class
- Collect as exit ticket

**Display Student Thinking:**
- Type up powerful examples
- Post on "Growth Wall"
- Celebrate changed thinking

**Long-Term Tracking:**
- Students keep running list
- Review at semester end
- Portfolio evidence of growth
