# Reading Strategy Tools

Tools that teach students HOW to read strategically.

## Tool Overview

| Tool | When | Purpose |
|------|------|---------|
| Reading Process Bookmark | Every reading | Before/during/after routine |
| Annotation Code Key | Every reading | Purposeful text marking |
| Before/During/After Protocol | First read | Comprehension monitoring |

---

## READING PROCESS BOOKMARK

**Format:** Laminated bookmark students keep in binder

### Content

**BEFORE READING:**
- [ ] Preview: Look at title, headings, images
- [ ] Prior Knowledge: What do I already know?
- [ ] Purpose: Why am I reading this?
- [ ] Predict: What will this be about?

**DURING READING:**
- [ ] Stop & Think: Pause every paragraph
  - What just happened?
  - What does this mean?
- [ ] Annotate: Mark up the text
- [ ] Monitor: Am I understanding? If not, REREAD

**AFTER READING:**
- [ ] Summarize: Main idea in 1-2 sentences
- [ ] Clarify: What's still unclear?
- [ ] Connect: How does this relate to my life/other texts?
- [ ] Reflect: What did I learn?

### Implementation

**Week 1:** Give each student bookmark, model with think-aloud
**Weeks 2-4:** Remind students, check boxes are being used
**Week 5+:** Students use automatically, spot-check occasionally

### Assessment
- Collect bookmarks weekly: Are boxes checked?
- Exit ticket: "Which step was most helpful?"
- Conference: "How does the bookmark help you?"

## ANNOTATION CODE KEY

**Format:** Small sticker inside binder cover

### The Code

**Comprehension Marks:**
- **!** = Surprising or Interesting
- **?** = Confusing or I Need Help
- **\*** = Important to Remember
- **⊕** = Main Idea / Central Claim

**Analysis Marks:**
- **↑** = Evidence / Quote I Can Use
- **□** = Vocabulary Word to Define
- **→** = Cause & Effect
- **≈** = Author's Purpose Shown Here

**Response Marks:**
- **♡** = I Agree / I Relate
- **✗** = I Disagree / I Question This
- **💡** = New Idea / Aha Moment
- **➜** = Connection to Another Text/My Life

### Color-Coding Alternative

- **Yellow** = Main ideas / Important info
- **Green** = Evidence / Quotes to use
- **Blue** = Vocabulary / Confusing words
- **Orange** = Author's purpose / rhetoric
- **Pink** = Questions / Things I don't understand

### Quality Indicators

**Good Annotation:**
- Multiple symbol types used
- Concentrated around important passages
- Margin notes explaining WHY
- Can explain choices

**Poor Annotation:**
- Everything highlighted (no discrimination)
- Random marks with no purpose
- No margin notes
- Can't explain why marked

## BEFORE/DURING/AFTER PROTOCOL

**Format:** 3-column graphic organizer
**Use:** First read of every major text (Days 1-2)

### The Organizer

| BEFORE Reading | DURING Reading | AFTER Reading |
|----------------|----------------|---------------|
| I predict this text is about... | Important quotes/ideas: | Main idea: |
| I already know... | 1. | New learning: |
| Questions I have: | 2. | Still confused about: |
| | 3. | Connection to my life: |

### Implementation

**BEFORE (3-4 min):**
- Complete left column FIRST
- Look at title and first paragraph only
- Share predictions with partner

**DURING (10-12 min):**
- Read text, pause to record important quotes
- After each section: Write one important idea
- Goal: 3-5 quotes/ideas by end

**AFTER (3-4 min):**
- Complete right column
- What was main idea? What did you learn?
- What's still confusing?

### Metacognitive Prompts

Ask students:
- "How did your prediction compare to what you read?"
- "What helped you understand?"
- "When did you get confused? What did you do?"
- "How does prior knowledge connect to this text?"

## Troubleshooting

### "Students aren't using the tools"
- Make tools permanent (laminate, glue into notebooks)
- Require use (can't submit without)
- Model value constantly
- Build into routine

### "Students say tools are 'extra work'"
- "These tools make learning EASIER"
- Show data: Students who use tools score higher
- Compare: 5 min with bookmark vs 30 min confused
