# Station-Based Activities

Active learning protocols with movement, visual creation, and peer interaction.

## Protocol Selection

| Protocol | Time | Best For | Day |
|----------|:----:|----------|:---:|
| Gallery Walk | 20-25 min | Visual learning, peer feedback | Days 3-4 |
| Annotation Stations | 20-25 min | Close reading, multiple text passes | Days 1-2 |

---

## GALLERY WALK

**Time:** 20-25 minutes | **Grouping:** Small groups (3-4)

### Phase 1: Create (10-12 min)

**Setup:**
- Groups of 3-4 students
- Each group gets chart paper + markers
- Timer: 12 minutes to create visual

**Student Roles:**
- **Recorder** - Writes on chart
- **Reader** - Finds evidence in text
- **Designer** - Organizes visual layout
- **Timekeeper** - Watches clock

**Chart Requirements:**
- Text evidence (quotes with ¶ numbers)
- Labels and explanations
- Visuals (thinking maps, diagrams)
- Group member names

**Task Examples:**

*Days 1-2:*
- "Create thinking map showing author's purpose with 5 evidence examples"
- "Design poster analyzing theme development across text"

*Days 3-4:*
- "Find 5 rhetorical devices - create visual explaining each"
- "Map author's argument structure (claim, evidence, reasoning)"

*Days 5-6:*
- "Compare two texts - Venn diagram with evidence"
- "Create timeline of how author's ideas develop"

### Phase 2: Gallery Walk (8-10 min)

**Setup:**
- Post charts around room
- Each student gets sticky notes (3 colors)
- Timer: 8 minutes to view all charts

**Feedback System:**
- **Green** = "Strong evidence" or "Great analysis"
- **Yellow** = "Question: Why did you...?" or "Can you explain...?"
- **Red** = "This could be stronger if..."

Leave 2-3 sticky notes per chart.

**Feedback Frames:**
- Green: "Your evidence from ¶___ is strong because..."
- Yellow: "I'm confused about... Can you explain?"
- Red: "You could strengthen this by adding..."

### Phase 3: Debrief (3-5 min)
- Return to your chart
- Read all feedback received
- Discuss: "What patterns? What should we revise?"

### Thinking Map Options

- **Circle Map** - Context (background info)
- **Tree Map** - Categorizing (types of evidence, themes)
- **Bubble Map** - Describing (author's tone, traits)
- **Flow Map** - Sequence (plot, argument structure)
- **Multi-Flow Map** - Cause/Effect (author choices, effects)
- **Brace Map** - Part-to-whole (text structure)

### Materials
- Chart paper (one per group)
- Markers (assorted colors)
- Sticky notes (3 colors)
- Tape for posting
- Task cards (one per group)
- Thinking map templates

## ANNOTATION STATIONS

**Time:** 20-25 minutes | **Grouping:** Individual (rotating)

### Setup
- 4-5 stations around classroom
- Each station has different annotation focus
- Students rotate every 4-5 minutes
- Result: Text annotated with multiple layers

### Station Designs

**Station 1: Vocabulary Hunt** (DOK 1-2)
- Highlight 5 unknown words
- Define using context clues
- Materials: Yellow highlighter, vocabulary worksheet

**Station 2: Purpose Tracker** (DOK 2-3)
- Find 3 sentences showing author's purpose
- Annotate WHY this shows purpose
- Materials: Orange highlighter, sticky notes

**Station 3: Rhetoric Analyzer** (DOK 3)
- Identify 3 rhetorical devices
- Label: R = repetition, M = metaphor, RQ = rhetorical question
- Write effect: "This makes the reader feel..."
- Materials: Colored pencils, annotation symbols

**Station 4: Evidence Collector** (DOK 2-3)
- Find strongest evidence for author's claim
- Highlight in green
- Rank: Which is strongest? Why?
- Materials: Green highlighter, evidence sheet

**Station 5: Structure Mapper** (DOK 2-3)
- Draw brackets around sections: intro, claim, evidence, conclusion
- Label each in margin
- Materials: Brackets, labels

### Rotation Protocol

**Timing:**
- 5 minutes per station
- Timer rings → rotate clockwise
- 25 minutes total (5 stations × 5 min)

**Teacher Role:**
- Circulate between stations
- Check work quality
- Prompt struggling students
- Keep strict time

**Debrief:**
- "Look at your fully annotated text!"
- "Which station was most helpful?"
- "How will these annotations help your organizer?"

### Differentiation

**Struggling:** Start at Station 1, may only complete 1-3, provide sentence frames
**Advanced:** Start at Station 5, add complexity, create additional tasks
**ELL:** Extended Station 1 time, bilingual dictionary, native language frames

### Materials
- Text copies (one per student)
- Highlighters (multiple colors)
- Station task cards (laminated, reusable)
- Sticky notes
- Visible timer
- Annotation code reference at each station
