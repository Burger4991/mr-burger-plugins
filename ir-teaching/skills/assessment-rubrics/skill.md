# Assessment Rubrics for 10th Grade Intensive Reading

## Purpose

Generate rubrics aligned to Florida BEST ELA achievement level descriptors (Levels 2-5) for any assessment type used in 10th grade Intensive Reading. Each rubric connects specific criteria to performance levels, student self-assessment checkboxes, and teacher feedback sections.

**Key principle**: Rubrics are transparent teaching tools shared with students at the start of the unit—not just grading instruments used at the end.

## Important Note: RACE vs CER

**RACE (Restate, Answer, Cite, Explain)** is the DEFAULT constructed response framework for 10th Grade Intensive Reading. It has four components:
- **R** (Restate): Restate the question as a statement
- **A** (Answer): Direct response to the question
- **C** (Cite): Text evidence with paragraph numbers
- **E** (Explain): Connection showing HOW evidence supports the answer

**ACE (Answer, Cite, Explain)** is the SHORTENED form of RACE used for exit tickets and quick checks — it drops the Restate step.

**CER (Claim, Evidence, Reasoning)** is the ALTERNATIVE framework used ONLY for argument-heavy units (ELA.10.R.2.4).

**Three-tier progression (applies to both RACE and ACE):**
- **Lite** (Units 1-3): Restate + Answer only (RACE Lite) or Answer only (ACE Lite)
- **Standard** (Units 4-6): Restate + Answer + Cite (RACE Standard) or Answer + Cite (ACE Standard)
- **Full** (Units 7+): Complete RACE or ACE with all components

Each unit chooses ONE framework: RACE (default for most units) or CER (for argument units). The rubric tiers apply consistently to whichever framework is selected.

**CRITICAL — Tier Enforcement Rule:**
When building a rubric, ONLY include criteria rows for the components in the current tier. Do NOT include all 4 RACE rows regardless of tier:
- **Lite tier rubric** → 2 rows only: R (Restate) + A (Answer). Do NOT include C or E rows.
- **Standard tier rubric** → 3 rows only: R (Restate) + A (Answer) + C (Cite). Do NOT include E row.
- **Full tier rubric** → 4 rows: R + A + C + E (all components).

If the teacher specifies a tier, the rubric MUST match that tier exactly. Including extra components confuses students about expectations and undermines the gradual release of the CR framework across the year.

## Rubric Types

Build rubrics for these assessment types used in the 6-day cycle:

### 1. ACE Response Rubric (4-point scale)
Most common written response format. Assesses the three components: Answer, Cite, Explain (or Restate, Answer, Cite, Explain if using RACE framework).

### 2. Graphic Organizer Rubric (3-point scale)
Grades completeness, evidence quality, and analysis depth in I Do / We Do / You Do organizers.

### 3. Multiple Choice Justification Rubric (2-point scale)
Scores correct answer selection + quality of STOP protocol explanation.

### 4. Discussion/Participation Rubric (3-point scale)
For Teacher-Led Rotation verbal responses. Assesses evidence use and reasoning in spoken format.

### 5. Exit Ticket Rubric (3-point scale)
Quick formative checks. Simplified scale (Complete/Partial/Incomplete) for daily use.

## Achievement Level Alignment

All rubrics must map to Florida BEST achievement levels (2-5):

| Level | Performance Descriptor | Rubric Interpretation |
|-------|----------------------|----------------------|
| **Level 5: Advanced** | Exceeds standard, nuanced analysis, multiple pieces of evidence, evaluates or compares with sophistication | 8-9 points (ACE) / 9-10 points (Organizer) |
| **Level 4: Proficient** | Meets standard, clear analysis, sufficient textual evidence, follows framework correctly | 6-7 points (ACE) / 7-8 points (Organizer) |
| **Level 3: Approaching** | Partial understanding, weak evidence or incomplete analysis, missing some components | 4-5 points (ACE) / 5-6 points (Organizer) |
| **Level 2: Beginning** | Minimal understanding, no/irrelevant evidence, incomplete or missing major components | 0-3 points (ACE) / 0-4 points (Organizer) |

**Important**: Pull descriptor language from the specific benchmark being assessed. For example, when scoring a theme rubric, use language from the theme benchmark achievement levels (see `benchmarks` skill, `standards/theme.md`). Each benchmark guide contains 5 achievement level descriptors—adapt those specific criteria into rubric language.

## Rubric Template Structure

Every rubric should include these sections:

### Header Information
- Benchmark number and clarification text
- Assessment name and date
- Student name and period
- Total points possible

### Criteria Rows
- **What**: What is being assessed (e.g., "Restate the Question")
- **How to spot it**: Observable behaviors or characteristics

### Level Descriptors (Columns)
- **Level 2** (0 points): Missing or minimal
- **Level 3** (1 point): Partial or approaching
- **Level 4** (2 points): Clear or proficient
- **Level 5** (3 points): Advanced or sophisticated
*(Point values vary by rubric type)*

### Scoring Section
- Point value for each criterion
- Total score
- Achievement Level mapping (score range → Level 2-5)

### Student Self-Assessment Column
Checkbox: "I think I'm at Level ___"
Students circle the level they believe they achieved BEFORE submitting.

### Teacher Feedback Section
- Space for 1-2 specific strengths
- Space for 1-2 areas for revision
- Next steps or action plan

## ACE Response Rubric (Full Template & Example)

### Template Structure

**Rubric**: ACE Response (Text-Based Constructed Response)

**Benchmark**: [Insert code, e.g., ELA.10.R.1.2 - Theme Analysis]

**Clarification**: [State what students are analyzing, e.g., "Analyze how the author develops universal themes"]

---

| Criterion | Level 2: Beginning (0 pts) | Level 3: Approaching (1 pt) | Level 4: Proficient (2 pts) | Level 5: Advanced (2 pts)* |
|-----------|---------------------------|---------------------------|--------------------------|--------------------------|
| **R - Restate the Question** | Missing restatement or completely off-topic | Restatement present but vague or incomplete; doesn't use key words from question | Restates question clearly with key words; creates clear topic sentence | Restates question with sophisticated rephrasing; incorporates benchmark language |
| **A - Answer the Question** | Missing or incorrect answer; no claim stated | Answer present but vague, unclear, or partially addresses question | Clear, direct answer that addresses the question; uses academic vocabulary | Answer is nuanced, specific, and addresses multiple aspects of the question |
| **C - Cite Text Evidence** | Missing or no text evidence; no citations | 1 piece of evidence present but lacks citation or is weakly relevant | 1-2 relevant quotes with paragraph numbers; quotes are brief and directly support answer | 2+ strong, varied quotes with accurate paragraph numbers; quotes are strategically chosen |
| **E - Explain the Connection** | Missing explanation; no reasoning | Explanation present but summarizes rather than analyzes; weak connection to evidence | Explains HOW evidence supports answer; uses reasoning verbs (demonstrates, reveals, illustrates) | Sophisticated analysis showing HOW evidence supports answer; explains WHY it matters for the benchmark |

**Scoring (scales by tier):**

**Full tier (R+A+C+E = 8 pts max):**
- 0-2 points: Level 2 (Beginning)
- 3-4 points: Level 3 (Approaching)
- 5-6 points: Level 4 (Proficient)
- 7-8 points: Level 5 (Advanced)

**Standard tier (R+A+C = 6 pts max):**
- 0-1 points: Level 2 (Beginning)
- 2-3 points: Level 3 (Approaching)
- 4-5 points: Level 4 (Proficient)
- 6 points: Level 5 (Advanced)

**Lite tier (R+A = 4 pts max):**
- 0-1 points: Level 2 (Beginning)
- 2 points: Level 3 (Approaching)
- 3 points: Level 4 (Proficient)
- 4 points: Level 5 (Advanced)

*Note: For Full tier, Level 5 on Explain criterion is worth 3 points in advanced responses to accommodate sophisticated multi-sentence reasoning; this creates max 9-point scale for exceptional responses.

### Concrete Example: Theme ACE Rubric

**Benchmark**: ELA.10.R.1.2 - Analyze and compare universal themes

**Text**: "The Story of an Hour" by Kate Chopin

**Question**: How does the author develop the theme of freedom vs. confinement in the story?

---

| Criterion | Level 2 | Level 3 | Level 4 | Level 5 |
|-----------|---------|---------|---------|---------|
| **R - Restate** | No restatement | "This story is about freedom." | "The author develops the theme of freedom vs. confinement..." | "Through Mrs. Mallard's transformation, the author develops the universal theme of freedom vs. confinement..." |
| **A - Answer** | "The character learns something" | "The theme is about being free" | "The author develops the theme that true freedom requires both physical and emotional independence" | "The author develops the theme that true freedom requires emotional independence—and its loss can be more devastating than never having experienced it" |
| **C - Cite** | No quotes or quote without citation | 1 quote present but missing paragraph number or weakly relevant, e.g., "She was in her room" | 1-2 relevant quotes with citations, e.g., "Free! Body and soul free!" (para 16) and "a long procession of years that would belong to her absolutely" (para 13) | 2+ strategically varied quotes showing theme development across text: "quite motionless" (para 4 - confinement), "Free! Body and soul free!" (para 16 - freedom realization), "joy that kills" (para 23 - freedom's loss) |
| **E - Explain** | "This shows freedom is important" | "This quote shows she wanted to be free. She felt trapped by her husband" | "This demonstrates that the author uses Mrs. Mallard's stillness in paragraph 4 to show her initial confinement, and her whispered declaration in paragraph 16 to show her newfound freedom. The contrast reveals the theme" | "These passages reveal that the author develops the theme through irony: true freedom emerges only when confinement ends, yet Mrs. Mallard cannot sustain this freedom. The metaphor of joy 'killing' her suggests that for women of her era, freedom itself is dangerous—more deadly than confinement. This develops the universal message that freedom requires not just the absence of constraints, but the strength to live autonomously" |

## Graphic Organizer Rubric (Full Template & Example)

### Template Structure

**Rubric**: Graphic Organizer Assessment

**Organizer Type**: [4-row Gradual Release / T-Chart / Venn Diagram / etc.]

**Benchmark**: [Insert code and topic]

**Clarification**: [State what students are organizing, e.g., "Track evidence of theme development across the text"]

---

| Criterion | Level 2 (0 pts) | Level 3 (1 pt) | Level 4 (2 pts) | Level 5 (3 pts) |
|-----------|-----------------|----------------|-----------------|-----------------|
| **Completeness** | Multiple sections blank or missing; organizer is incomplete | Most sections filled but some missing; may have vague or one-word responses | All sections completed with full responses; clear and legible | All sections completed thoroughly; demonstrates effort and thoroughness |
| **Evidence Quality** | No text evidence or evidence is irrelevant/generic (e.g., "sad") | Some text evidence present but lacks paragraph numbers or is vaguely relevant | Specific quotes with paragraph numbers; directly supports claims/analysis | Multiple specific quotes with paragraph numbers; strategically chosen to show development/progression |
| **Analysis Depth** | Missing analysis or analysis column blank | Analysis present but mostly summarizes events rather than explains connections | Explains HOW evidence supports claim; uses reasoning verbs; shows understanding of benchmark | Sophisticated analysis showing WHY evidence matters; connects to larger patterns or theme; evaluates significance |
| **Scaffold Removal (You Do row)** | You Do row copies I Do row or is incomplete | You Do row shows some independent thinking but may paraphrase I Do closely | You Do row is clearly independent; different evidence/perspective than I Do; maintains quality | You Do row demonstrates confident independent thinking; complex ideas; extends beyond I Do model |

**Scoring:**
- **0-4 points**: Level 2 (Beginning)
- **5-6 points**: Level 3 (Approaching)
- **7-8 points**: Level 4 (Proficient)
- **9-10 points**: Level 5 (Advanced)

**Benchmark**: ELA.10.R.1.2 - Analyze and compare universal themes

**Text**: "The Story of an Hour" by Kate Chopin

**Organizer**: Single Theme Deep Dive (Beginning / Middle / End Evidence)

---

| Criterion | Level 2 | Level 3 | Level 4 | Level 5 |
|-----------|---------|---------|---------|---------|
| **Completeness** | Sections blank or 1-word responses; "Beginning Evidence" blank | Most columns filled but some vague; "Middle Evidence" column says "She realizes she's free" | All rows completed with 1-2 sentence responses; clear and legible throughout | All rows thoroughly completed with detailed responses; demonstrates understanding throughout |
| **Evidence Quality** | No quotes or vague description: "She was quiet" | Some quotes but missing paragraph #: "She said 'Free! Body and soul free!'" OR only topic-related: "She didn't like her husband" | 1-2 quotes per section with paragraph numbers: "Free! Body and soul free!" ## Multiple Choice Justification Rubric

### Template Structure

**Rubric**: Multiple Choice + STOP Analysis

**Benchmark**: [Insert code]

**Clarification**: Students select correct answer AND explain why using STOP protocol (Step 3 of question annotation strategy). Students label which choice is Silly, Tricky, Opposite, Proven and justify each label.

---

| Criterion | Level 2 (0 pts) | Level 3 (1 pt) | Level 4 (2 pts) |
|-----------|-----------------|----------------|-----------------|
| **Correct Answer Selection** | Incorrect answer selected; no understanding shown | Answer selected but without justification or with weak reasoning | Correct answer selected with clear justification |
| **STOP Protocol Application** (Step 3 of Question Annotation) | Missing STOP labels or all incorrect; no analysis shown | Some STOP labels present; 2-3 correctly identified | All 4 choices correctly labeled (Silly, Tricky, Opposite, Proven); accurate reasoning for each |

**Scoring:**
- **0 points**: Level 2 (Beginning)
- **1-2 points**: Level 3 (Approaching)
- **3-4 points**: Level 4 (Proficient)

**Total**: _____ / 4 points → Achievement Level: _____

---

### Concrete Example

**Question**: What theme is developed through Phineas's loyalty to Gene?

A. Friendship requires sacrifice.
B. Jealousy destroys relationships.
C. Competition leads to success.
D. Betrayal is inevitable.

---

| Criterion | Student Response |
|-----------|------------------|
| **Answer** | Selected: A (Correct) |
| **STOP Analysis** | **A (Proven)**: This is the correct answer. Phineas shows loyalty throughout the novel, and his friendship with Gene requires him to sacrifice—he even accepts Gene's guilt when Gene pushes him off the tree. This choice directly proves the theme. |
| | **B (Tricky)**: Jealousy is mentioned in the novel, but it's not the primary theme. This answer is accurate in that jealousy does cause some tension, but it misses the larger message about friendship and sacrifice. |
| | **C (Opposite)**: The novel doesn't focus on competition leading to success; in fact, it shows that trying to compete with Phineas leads to conflict, not success. This contradicts the actual theme. |
| | **D (Silly)**: While betrayal happens, the novel ultimately shows forgiveness and loyalty, not inevitability of betrayal. This is too negative and doesn't match the text. |

**Teacher Comments**: Strong STOP analysis. You correctly identified why A is proven and why B is tricky—that's where many students struggle. Next time, try to cite a specific scene (e.g., "when Phineas forgives Gene after the accident") to make your reasoning even stronger.

**Achievement Level**: Proficient (4 points)

## Discussion/Participation Rubric (Verbal Responses)

### Template Structure

**Rubric**: Teacher-Led Rotation Discussion

**Benchmark**: [Insert code]

**Clarification**: Students verbally explain ideas during small-group rotation. Teacher records on checklist as student speaks.

---

| Criterion | Level 2 (0 pts) | Level 3 (1 pt) | Level 4 (2 pts) |
|-----------|-----------------|----------------|-----------------|
| **Claim/Idea Clarity** | Unclear or off-topic response; difficult to understand | Idea stated but vague or partially addresses the question | Clear, direct response that answers the question asked |
| **Use of Evidence** | No text evidence; relies on opinion | References text but without citation ("He was sad") | Cites specific text with reference ("In paragraph 5, the author writes...") |
| **Reasoning Quality** | No explanation; just states a fact | Attempts explanation but mostly summarizes ("This shows he's sad") | Explains HOW evidence supports claim; uses reasoning language |

**Scoring:**
- **0 points**: Level 2 (Beginning)
- **1-2 points**: Level 3 (Approaching)
- **3-4 points**: Level 4 (Proficient)

**Total**: _____ / 4 points → Achievement Level: _____

## Exit Ticket Rubric (Simplified 3-Point Scale)

### Template Structure

**Rubric**: Exit Ticket (Quick Formative Check)

**Purpose**: Daily formative assessment using single ACE question or focused prompt. Simplified scale for quick grading and feedback.

---

| Score | Descriptor | What to Do Next |
|-------|-----------|-----------------|
| **3 - Complete** | Response answers the question with textual evidence and reasoning. Student demonstrates understanding of the day's skill. | Praise specific strength. Student is ready to move forward. |
| **2 - Partial** | Response attempts to answer the question but evidence is weak, missing citation, or reasoning is incomplete. Student is close but not quite there yet. | Provide specific feedback about what's missing. Include in Day 4 small-group review. |
| **1 - Incomplete** | Response missing key components (no answer, no evidence, no reasoning). Student does not yet demonstrate understanding. | Flag for Day 4-5 intervention. Include in small-group intensive session. May need organizer review or sentence frames. |

---

### Concrete Example

**Exit Ticket - Day 2**

**Benchmark**: ELA.10.R.1.2 (Theme)

**Prompt**: Identify one universal theme from today's reading and explain how the author introduces it.

---

**Student Response**:
"The theme of perseverance appears when Marcus tries three times to solve the puzzle. In paragraph 8, it says 'After three failed attempts, Marcus finally succeeded.' This shows the theme because Marcus doesn't give up, which is what perseverance means."

**Scoring**: 2 (Partial)

**Strengths**: You identified a clear theme and found a relevant quote with the paragraph number. Good job!

**Revision Needed**: You stated what perseverance means, but I need you to explain HOW the author introduces this theme. Does the author introduce it through dialogue? A character's actions? A conflict? Revise to explain the METHOD the author uses.

**Next Step**: Include this in your Day 4 organizer work. We'll review how authors introduce themes before you write your ACE response.

## Scaffolding for Student Self-Assessment

Self-assessment is not just a box students check—it's a thinking tool that makes them aware of expectations and accountable to standards.

### Days 1-2: Teacher Models, Student Observes
- Teacher presents blank rubric on document camera
- Teacher reads a sample student response aloud
- Class discusses: "What level is this response? What tells us?"
- Teacher scores while students watch, explains reasoning
- Students highlight/annotate which descriptors match the response
- **Not ready for independent self-assessment yet—building vocabulary**

### Days 3-4: Teacher Scores, Student Labels
- Teacher scores organizers or responses using rubric
- Students receive rubric with teacher's score
- Students match the score to the rubric descriptor and write: "My teacher scored this 2 points because..."
- Class discusses why specific scores were given
- **Students learning to recognize quality; seeing rubric language in action**

### Days 5-6: Student Self-Assesses, Teacher Validates
- Before submitting, students self-assess using rubric checklist
- Students circle: "I think I'm at Level _____"
- Students write brief explanation: "I chose Level ___ because..."
- Teacher scores independently, then compares to student self-assessment
- Teacher conference: "You said Level 3. I scored you Level 4 because... Here's what moved you to proficient"
- **Students own the assessment; teacher validates accuracy**

### Week 2+: Student Self-Assesses Independently
- Rubric serves as study guide students review before writing
- Students self-assess without teacher input
- Rubric becomes internalized—"I can hear the rubric in my head while I write"
- Teacher spot-checks accuracy but focuses on content feedback

## ESOL Modifications for Rubrics

**For ESOL Levels 1-3 students**, adapt rubrics to support language development:

### Simplified Language
- Replace "sophisticated," "nuanced," "conveys" with "clear," "detailed," "shows"
- Use shorter descriptor sentences
- **Example**: Instead of "Analysis revealing author's intent through strategic device selection," use: "Explains how the author uses the device"

### Visual Rubrics
Create visual versions with:
- Example responses at each level (not just descriptions)
- Color-coding (Green = Level 4, Yellow = Level 3, Red = Level 2)
- Icons or images representing each criterion
- **Example**: Thinking bubble icon = "Reasoning column," Quote mark icon = "Evidence column"

### Flexible Evidence Requirements
- Allow evidence citation in L1 if text comprehension is demonstrated (e.g., Spanish quote from bilingual text, drawing-based response)
- Reduce minimum evidence pieces (1 quote instead of 2 for Meets level)
- Allow sentence frames to remain through Day 6 if student is at ESOL Level 1-2

### Think-Pair-Share Before Writing
- Students verbally explain their response to a partner using rubric language first
- Peer gives feedback using rubric categories
- Student writes after verbal rehearsal
- **This bridges oral language to written analysis**

### Word Bank on Rubric
Include key vocabulary students need:
- For Reasoning: demonstrates, reveals, illustrates, shows, explains, because
- For Theme: universal, develop, introduce, support, evidence, analyze
- For Comparison: similar, different, both, contrast, alike

## Benchmark-Specific Rubric Language

When creating a rubric, **pull exact language from the benchmark's achievement level descriptors**. Each benchmark guide contains this information.

### Example: Using Benchmark Language

**Benchmark**: ELA.10.R.1.2 (Theme) Achievement Levels (from `standards/theme.md`)

| Level | Benchmark Language |
|-------|-------------------|
| Level 5 | "Analyzes AND evaluates how universal themes are developed" |
| Level 4 | "Analyzes how universal themes are developed throughout text" |
| Level 3 | "Identifies universal themes in text" |
| Level 2 | "Identifies topics but may confuse with themes" |

**How to incorporate into ACE rubric:**
- Level 5 Answer cell: "Claim analyzes AND evaluates theme development (e.g., compares themes or discusses author's choices)"
- Level 4 Answer cell: "Claim analyzes how theme is developed throughout text"
- Level 3 Answer cell: "Identifies a universal theme with some analysis"
- Level 2 Answer cell: "Identifies a topic or concept but may confuse with universal theme"

**Key principle**: Never write vague rubric language like "good understanding." Always tie to the specific benchmark's achievement level criteria.

## Common Pitfalls to Avoid

❌ **Using identical descriptors across all levels** → Students can't distinguish quality differences

❌ **Being too vague** ("Good" / "Okay" / "Needs work") → Doesn't connect to benchmark standards

❌ **Forgetting student self-assessment** → Rubric becomes a grading tool, not a learning tool

❌ **Not sharing rubric until assessment day** → Students don't know expectations while learning

❌ **Making rubric columns too narrow** → Teachers can't write legible feedback; scores crowd together

❌ **Using inconsistent point scales** → Confuses students (sometimes 4-point, sometimes 3-point)

❌ **Copying achievement levels without adapting** → Generic language that doesn't match the specific benchmark

## Rubric Consistency Across Units

To help students build assessment literacy and apply rubrics fluently:

### Keep ACE/RACE Rubric Structure Consistent
- Number of criteria rows matches the current tier: Lite = 2 rows (R+A), Standard = 3 rows (R+A+C), Full = 4 rows (R+A+C+E)
- Always 0-2 points per criterion (max points scale with tier: Lite = 4 pts, Standard = 6 pts, Full = 8 pts)
- Always same color/formatting per tier level
- **Result**: By Unit 3, students know the rubric structure; new rows are added gradually as tiers advance

### Adapt Organizer Rubric by Type
- Single-theme organizer uses same 4 criteria (Completeness, Evidence Quality, Analysis, Scaffold Removal)
- Comparison organizer adapts Criterion 2 to "Comparison Clarity"
- **Result**: Students see familiar structure but adapted to the task

### Keep Exit Ticket Scale Simple
- Always 3 points (Complete/Partial/Incomplete)
- Always one question
- Always same format (brief prompt)
- **Result**: Students can self-assess exit tickets quickly without rereading criteria
