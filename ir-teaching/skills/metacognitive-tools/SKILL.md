# METACOGNITIVE TOOLS & ROUTINES
**Teaching Students HOW to Learn**

---

## PURPOSE

These tools teach students to think about their thinking - to monitor comprehension, evaluate their own work, track growth, and take ownership of learning. They address fundamental skill gaps:
- Students don't know HOW to read strategically
- Students don't recognize what quality work looks like
- Students can't self-assess or revise effectively
- Students don't connect effort to success

**Key Principle:** Before students can be independent learners, they must develop metacognitive awareness - the ability to monitor and regulate their own learning processes.

## TOOL 1: READING PROCESS BOOKMARK

### Overview
**Purpose:** Establish consistent before/during/after reading routine
**Format:** Laminated bookmark (students keep in binder)
**Frequency:** Use every time students read any text

### What's On It

**BEFORE READING:**
- Preview: Look at title, headings, images
- Prior Knowledge: What do I already know about this topic?
- Purpose: Why am I reading this? What should I learn?
- Predict: What do I think this will be about?

**DURING READING:**
- Stop & Think: Pause every paragraph
  - What just happened?
  - What does this mean?
- Annotate: Mark up the text (see Annotation Code)
- Monitor: Am I understanding? If not, REREAD

**AFTER READING:**
- Summarize: Main idea in 1-2 sentences
- Clarify: What's still unclear? How can I find out?
- Connect: How does this relate to my life, other texts, the world?
- Reflect: What did I learn? What surprised me?

### Implementation

**Week 1: Introduce**
- Give each student laminated bookmark
- Model using it with a short text (think-aloud)
- "Watch me check off each box as I read"
- Students practice with partner

**Week 2-4: Guided Practice**
- Remind students: "Use your bookmark for today's reading"
- Check: "Did you complete the BEFORE section?"
- Circulate: Verify students using it

**Week 5+: Independent Use**
- Students use automatically (habit established)
- Spot-check occasionally
- "Show me your bookmark - what did you check off?"

### Assessment Integration
- Collect bookmarks weekly: Are boxes checked off?
- Exit ticket: "Which step was most helpful today?"
- Conference: "How does the bookmark help you understand better?"

## TOOL 2: ANNOTATION CODE KEY

### Overview
**Purpose:** Make annotation purposeful (not random highlighting)
**Format:** Small sticker/sticky note inside binder cover
**Frequency:** Reference every time students annotate text

### The Code

**COMPREHENSION MARKS:**
- **!** = Surprising or Interesting
- **?** = Confusing or I Need Help
- **\*** = Important to Remember
- **+** = Main Idea / Central Claim

**ANALYSIS MARKS:**
- **^** = Evidence / Quote I Can Use
- **[ ]** = Vocabulary Word to Define
- **->** = Cause & Effect
- **~** = Author's Purpose Shown Here

**RESPONSE MARKS:**
- **<3** = I Agree / I Relate
- **X** = I Disagree / I Question This
- **!** = New Idea / Aha Moment
- **>>** = Connection to Another Text/My Life

### Color-Coding Alternative

If students prefer highlighters to symbols:
- **Yellow** = Main ideas / Important info
- **Green** = Evidence / Quotes to use
- **Blue** = Vocabulary / Confusing words
- **Orange** = Author's purpose / rhetoric
- **Pink** = Questions / Things I don't understand

### Implementation

**Week 1: Teach the Code**
- Project annotation key on board
- Model: Read paragraph aloud, annotate with think-aloud
- "I'm putting a star here because this is the main idea..."
- "I'm putting an arrow here because I can use this as evidence..."

**Week 2-4: Practice Together**
- Distribute copies of code (sticky note for binder)
- Students annotate text in class (you check)
- "Show me your annotations - what symbols did you use?"
- Provide feedback: "Good! Now add more evidence markers"

**Week 5+: Expect Independence**
- Students annotate automatically
- Collect annotated texts periodically
- Grade for purposeful annotation (not random marks)

### Quality Indicators

**Good Annotation:**
- Multiple symbol types used
- Concentrated around important passages
- Notes written in margins explaining WHY
- Can explain choices: "I starred this because..."

**Poor Annotation:**
- Everything highlighted (no discrimination)
- Random marks with no purpose
- No margin notes/explanations
- Can't explain why they marked something

## TOOL 3: BEFORE/DURING/AFTER THINKING PROTOCOL

### Overview
**Purpose:** Monitor comprehension throughout reading process
**Format:** 3-column graphic organizer
**Frequency:** Use for first read of every major text (Days 1-2)

### The Organizer

| BEFORE Reading | DURING Reading | AFTER Reading |
|----------------|----------------|---------------|
| **I predict this text is about...** | **Important quotes/ideas:** | **Main idea:** |
| **I already know...** | 1. | **New learning:** |
| **Questions I have:** | 2. | **Still confused about:** |
| | 3. | **Connection to my life:** |

### Implementation

**BEFORE Reading (3-4 minutes):**
- Students complete left column FIRST (before seeing text)
- "Look at the title and first paragraph only"
- "What do you predict? What do you already know?"
- Share predictions with partner

**DURING Reading (10-12 minutes):**
- Students read text, pause to record important quotes
- After each section: "Write one important idea in middle column"
- Goal: 3-5 quotes/ideas by end of reading

**AFTER Reading (3-4 minutes):**
- Students complete right column (reflection)
- "What was the main idea? What did you learn?"
- "What's still confusing? How can you find out?"
- Share with partner or class

### Metacognitive Prompts

As students work, circulate and ask:
- "How did your prediction compare to what you actually read?"
- "What helped you understand this text?"
- "When did you get confused? What did you do about it?"
- "How does your prior knowledge connect to this text?"

### Assessment
- Collect organizers: Check for thoughtful responses
- Conference: "Walk me through your thinking process"
- Growth tracking: Compare early organizers (sparse) to later ones (detailed)

## TOOL 4: SELF-ASSESSMENT CHECKLIST

### Overview
**Purpose:** Students evaluate own work quality before submission
**Format:** Checklist attached to every major assignment
**Frequency:** Use before submitting organizers, CR, projects

### The Checklist

**CONTENT (Answering the Question):**
- Answer the ENTIRE question/prompt (not just part)
- Include required number of text evidence pieces
- Explain HOW evidence supports answer
- Use academic vocabulary correctly
- Stay on topic throughout

**WRITING QUALITY:**
- Complete sentences (no fragments)
- Correct spelling (used spell-check)
- Correct punctuation (capitals, periods, commas)
- Organized into paragraphs
- Used transition words

**EVIDENCE & ANALYSIS:**
- Every claim has text evidence
- Used quotation marks around quotes
- Cited where evidence came from (paragraph #)
- Explained WHY evidence matters

**EFFORT & PRIDE:**
- This is my BEST work (not rushed)
- I revised after reading it over
- I would be proud to share this
- I used class time wisely
- I asked for help when needed

**SELF-EVALUATION:**
- My self-assessment score: _____ / 10
- What I'm most proud of: ________________
- One thing I could improve: ________________

### Implementation

**Week 1: Model It**
- Project sample student work (anonymous)
- Go through checklist together as class
- "Does this work meet all criteria? Let's check..."
- Score it together: "What score would you give?"

**Week 2-4: Guided Practice**
- Students complete checklist before submitting work
- Teacher reviews: Does self-assessment match actual quality?
- Give feedback: "You said 8/10 but this is really 6/10 because..."
- Teach accurate self-evaluation

**Week 5+: Required Routine**
- No work accepted without attached checklist
- Students internalize standards (know what quality looks like)
- Self-assessments become more accurate over time

## TOOL 5: WEEKLY REFLECTION FORM

### Overview
**Purpose:** Track growth, set goals, build growth mindset
**Format:** Short reflection (5 minutes every Friday)
**Frequency:** Weekly ritual (last 5 min of class)

### The Form

**WEEKLY REFLECTION - Week of _______**

**This week I learned:**

**One thing I got better at:**

**One thing that's still hard:**

**My effort this week was:**
- Low (I could have tried harder)
- Medium (I did okay)
- High (I gave my best effort)

**My goal for next week:**

**Something my teacher should know:**

### Implementation

**Every Friday (Last 5 minutes):**
- Music playing (calm reflection time)
- Students complete form independently
- Brief share with partner (optional)
- Collect forms (teacher reads over weekend)

**Teacher Response:**
- Read all reflections over weekend
- Write 1-2 sentence encouraging comment on each
- Return Monday with feedback
- Follow up on student concerns/questions

**Student Portfolio:**
- Students keep reflections in folder/binder
- Review at midpoint/end of quarter
- "Look back - how have you grown?"
- Metacognitive awareness of learning journey

## TOOL 6: STRATEGY REMINDER POSTERS

### Overview
**Purpose:** Visual reference for problem-solving strategies
**Format:** Large posters on classroom walls
**Frequency:** Students reference when stuck

### Poster 1: How to Find Text Evidence

```
1. Read the question carefully - what is it asking?
2. Go back to the text - don't answer from memory
3. Find a sentence that DIRECTLY answers the question
4. Copy the evidence word-for-word (use quotation marks)
5. Explain WHY this evidence answers the question
```

### Poster 2: CER Writing Strategy

```
C = Claim (Answer the question in one sentence)
E = Evidence (Quote from text that proves your claim)
R = Reasoning (Explain HOW evidence proves your claim)
```

### Poster 3: When I'm Stuck... (Problem-Solving)

```
If I'm confused by a word -> Use context clues or ask
If I don't understand a paragraph -> Reread it slowly
If I don't know how to start -> Look at exemplar/model
If I'm stuck on a question -> Check the strategy poster
If I still need help -> Ask a classmate, then ask teacher
```

### Poster 4: STOP Test-Taking Strategy

```
S = STOP and read the question carefully (underline key words)
T = THINK about what the question is asking
O = OPTIONS - read all answer choices before selecting
P = PROVE your answer with evidence from the text
```

### Implementation

**Create Posters:**
- Use large poster board or chart paper
- Color-code for visual appeal
- Hang permanently on classroom walls
- Strategic placement (near where students work)

**Teach Students to Use:**
- Reference constantly: "Look at the CER poster - what's step 1?"
- Students point to posters when helping each other
- "Check the 'When I'm Stuck' poster before asking me"
- Make strategies visible and concrete

## TOOL 7: "I USED TO THINK... NOW I THINK..." PROTOCOL

### Overview
**Purpose:** Reflect on changed thinking and learning journey
**Format:** Sentence completion at end of unit
**Frequency:** Last day of every unit (Day 6)

### The Protocol

**Sentence Frame:**
"I used to think __________, but now I think __________ because __________."

**Examples:**

*About Content:*
"I used to think Obama was just giving a speech, but now I think he was carefully using rhetoric to persuade his audience because I noticed all the repetition and emotional language."

*About Skills:*
"I used to think annotation was just highlighting random stuff, but now I think it's about marking specific things like evidence and confusing parts because that helps me understand better."

*About Learning Process:*
"I used to think I wasn't good at reading, but now I think I can improve with practice because my scores went up when I used the strategies."

### Implementation

**Last Day of Unit (5 minutes):**
- Project sentence frame on board
- Students complete independently (write in notebook)
- Share with partner or class (volunteer shares)
- Collect as exit ticket

**Display Student Thinking:**
- Type up powerful examples
- Post on "Growth Wall" in classroom
- Celebrate changed thinking

## DAILY METACOGNITIVE ROUTINES

### Start of Class Routine (2-3 minutes)

**Metacognitive Priming:**
- "Today we're working on [skill]. You'll know you've learned it when you can [success criteria]."
- "Before we start, get out your Reading Process Bookmark"
- "What strategy did you use last class that helped you?"

### During Class Check-Ins (Ongoing)

**As You Circulate:**
- "What strategy are you using right now?"
- "When you got stuck, what did you do?"
- "How do you know if your answer is correct?"
- "What's helping you understand this?"

**NOT:**
- "Do you understand?" (closed question, unhelpful)
- Just giving answers without prompting thinking

### End of Class Routine (2-3 minutes)

**Exit Ticket with Metacognitive Focus:**
- "What did you learn today?"
- "What strategy helped you most?"
- "What's still confusing?"
- "On a scale of 1-10, how well do you understand [skill]? Why?"

## INTEGRATION WITH 6-DAY CYCLE

### Recommended Tool Use by Day

**Day 1:**
- Reading Process Bookmark (first read)
- Annotation Code Key (while reading)
- Before/During/After Protocol (comprehension)
- Strategy Posters (reference as needed)

**Days 2-4:**
- Annotation Code Key (ongoing analysis)
- Strategy Posters (CER, evidence finding)
- Self-Assessment Checklist (before submitting organizer)

**Days 5-6:**
- Strategy Posters (STOP, test-taking)
- Self-Assessment Checklist (before submitting CR)
- "I Used to Think..." Protocol (end of unit)

**Every Friday:**
- Weekly Reflection Form (growth tracking)

## TROUBLESHOOTING

### "Students aren't using the tools"

**Causes:**
- Tools not accessible (lost, forgotten)
- Don't see value yet
- Not built into routine

**Solutions:**
- Make tools permanent (laminate, glue into notebooks)
- Require use (can't submit work without checklist)
- Model value constantly ("The bookmark helped me understand...")
- Build into routine (use same tools same time every week)

### "Students say tools are 'extra work'"

**Reframe:**
- "These tools make learning EASIER, not harder"
- Show data: "Students who use checklist score 10% higher"
- Compare time: "5 minutes using bookmark vs. 30 minutes confused"
- It's like training wheels - feels awkward at first, then helps

### "I don't have time to add metacognitive routines"

**Minimum Viable Metacognition:**
- Start with JUST Reading Process Bookmark (Week 1)
- Add Self-Assessment Checklist (Week 3)
- Add Weekly Reflection (Week 5)
- Build slowly, not all at once

## INTEGRATION WITH AGENTS/UNIT BUILDING

### When Building Units (Phases 1-3)

**Phase 1: Unit Planning**
- Identify which metacognitive tools fit each day
- Plan routines: When will students use bookmarks? Checklists?

**Phase 2: Lesson Planning**
- Embed metacognitive prompts in lesson procedures
- Include tool references: "Students use Self-Assessment Checklist before submitting"
- Add metacognitive questions to teacher script

**Phase 3: Materials Building**
- Include checklist with rubric
- Provide bookmark reference in student packet
- Add "I Used to Think..." exit ticket for Day 6
