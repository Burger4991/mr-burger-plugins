# Literacy Skills × Benchmark Connections Matrix

## PURPOSE

This document shows the **explicit connections** between:
- **6 Core Literacy Skills** (Vocabulary, Comprehension, Close Reading, Writing, Test-Taking, Metacognition)
- **Florida BEST ELA Benchmarks** (Theme, Rhetoric, Argument, etc.)

**Core Principle:** Every benchmark requires multiple literacy skills working together. Students can't master a benchmark without the underlying literacy skills.

---

## THE 6 CORE LITERACY SKILLS (Quick Reference)

| # | Skill | What Students Do |
|---|-------|------------------|
| **1** | **Vocabulary Development** | Learn and use academic/domain-specific words |
| **2** | **Reading Comprehension** | Understand what they read; monitor comprehension |
| **3** | **Close Reading & Annotation** | Mark text purposefully; find evidence |
| **4** | **Analytical Writing** | Explain thinking with text evidence in writing |
| **5** | **Test-Taking Strategies** | Use STOP for MC, CER for CR |
| **6** | **Metacognition** | Think about their thinking; self-assess |

## BENCHMARK × LITERACY SKILLS MATRIX

### ELA.10.R.1.2 - THEME

**Standard:** Analyze and compare universal themes and their development

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know "theme," "universal," "development," "convey" | Day 1 Teacher-Led: Explicitly teach 4-5 theme vocabulary words (theme, universal, symbolism, motif) with visuals |
| **2. Comprehension** | Must understand plot to track theme | Day 1-2: Teach predicting/questioning strategies; first read with comprehension questions |
| **3. Close Reading** | Must find evidence of theme at multiple points in text | Day 2: Model annotation of theme evidence (mark symbols, character changes, pivotal moments) |
| **4. Analytical Writing** | Must explain how theme develops from beginning to end | Days 3-4: Gradual release with theme organizer (trace development with evidence + reasoning) |
| **5** Test-Taking** | MC: Distinguish theme from topic; CR: Write theme analysis paragraph | Day 5: Model STOP for theme MC ("Is this universal or too specific?"); Model CER paragraph |
| **6. Metacognition** | Must monitor: "Am I confusing theme with topic?" | All days: Use self-assessment checklist; common error reminders |

**Common Student Gap:** Confusing theme with topic ("The theme is love" instead of "Love requires sacrifice")

**Remediation:** Teach question test: "Is it universal? Is it a message? Can I prove it with evidence?"

### ELA.10.R.3.4 - RHETORIC

**Standard:** Analyze rhetorical devices and their effects in persuasive texts

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know rhetorical device names (repetition, parallelism, anaphora, ethos, pathos, logos) | Day 1-2 Teacher-Led: Teach 6-8 rhetoric terms with anchor chart examples |
| **2. Comprehension** | Must understand speaker's purpose and audience | Day 1: Teach "Who/What/When/Where/Why" of speech before analyzing rhetoric |
| **3. Close Reading** | Must identify specific examples of devices in text | Day 2: Annotation stations (device identifier station - mark repetition, questions, etc.) |
| **4. Analytical Writing** | Must explain device + effect + purpose connection | Days 3-4: Organizer with 3 columns (Device, Effect, Purpose) - explain HOW device achieves effect |
| **5. Test-Taking** | MC: Identify device and effect; CR: Analyze effectiveness | Day 5: Model STOP for rhetoric MC; Model timed paragraph analyzing cumulative effect |
| **6. Metacognition** | Must monitor: "Did I just NAME the device or also explain the EFFECT?" | Days 3-4: Self-assessment - "Did I explain effect? Did I connect to purpose?" |

**Common Student Gap:** Identifying device but not explaining effect

**Remediation:** Require sentence frame: "This [device] makes the audience feel/think/do _____ because _____."

### ELA.10.R.2.4 - ARGUMENT

**Standard:** Compare and evaluate opposing arguments

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know "claim," "evidence," "reasoning," "counterclaim," "rebuttal," "bias" | Day 1 Teacher-Led: Teach C.E.R. vocabulary + counterclaim/rebuttal terms |
| **2. Comprehension** | Must distinguish author's argument from opposing views | Day 1-2: Teach identifying claim vs evidence vs counterclaim (label each) |
| **3. Close Reading** | Must locate evidence and evaluate its strength | Day 2: Annotation - mark claim, evidence, counterclaim with different symbols |
| **4. Analytical Writing** | Must explain how evidence supports claim (reasoning) | Days 3-4: C.E.R. organizer - explicit reasoning practice ("This evidence proves ___ because ___") |
| **5. Test-Taking** | MC: Distinguish claim from evidence; CR: Write argument analysis | Day 5: Model STOP for argument MC ("Is this claim or evidence?"); Model CER for effectiveness |
| **6. Metacognition** | Must check: "Did I explain HOW evidence supports claim?" | Days 3-4: Self-assessment checklist specifically for reasoning completeness |

**Common Student Gap:** Confusing claim (opinion) with evidence (facts)

**Remediation:** Ask: "Is this debatable? If yes → claim. Is this provable? If yes → evidence."

### ELA.10.R.2.3 - AUTHOR'S PURPOSE & PERSPECTIVE

**Standard:** Analyze author's purpose and perspective in informational/historical texts

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know "purpose," "perspective," "bias," "tone," "inform," "persuade," "entertain" | Day 1: Teach purpose vocabulary (P.I.E. = Persuade/Inform/Entertain) + perspective/bias terms |
| **2. Comprehension** | Must understand what author is saying before analyzing why | Day 1-2: First read with main idea questions; identify author's position |
| **3. Close Reading** | Must find evidence of author's perspective (word choice, selection of details, tone) | Day 2: Annotate for bias indicators (loaded words, one-sided details, tone shifts) |
| **4. Analytical Writing** | Must explain how author's choices reveal purpose/perspective | Days 3-4: Organizer linking evidence (word choice, details) to purpose/perspective |
| **5. Test-Taking** | MC: Identify purpose and perspective; CR: Explain how perspective shapes text | Day 5: Model STOP for purpose/perspective questions; CER paragraph on bias/perspective |
| **6. Metacognition** | Must ask: "What's the author trying to make me think? What's left out?" | All days: Critical reading stance - "What perspective is NOT represented?" |

**Common Student Gap:** Identifying purpose but not recognizing perspective/bias

**Remediation:** Ask: "Whose side is the author on? What details did they emphasize? What did they leave out?"

### ELA.10.R.1.1 - LITERARY ELEMENTS

**Standard:** Analyze how character, setting, and plot interact to create meaning

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know "characterization," "setting," "plot," "conflict," "interaction," "contribute" | Day 1: Teach literary elements vocabulary (direct/indirect characterization, types of conflict) |
| **2. Comprehension** | Must understand plot events and character development | Day 1-2: Story map or plot diagram; character trait tracking |
| **3. Close Reading** | Must find evidence of how elements interact (not just identify them) | Day 2: Annotate moments where setting impacts character OR character drives plot |
| **4. Analytical Writing** | Must explain how interaction creates meaning | Days 3-4: Organizer with "How does X impact Y? What meaning does this create?" |
| **5. Test-Taking** | MC: Analyze interaction and meaning; CR: Explain contribution to overall meaning | Day 5: Model STOP for interaction questions; CER for meaning contribution |
| **6. Metacognition** | Must check: "Did I show INTERACTION or just identify elements?" | Days 3-4: Self-check - "Did I explain how they work TOGETHER?" |

**Common Student Gap:** Identifying elements separately without showing interaction

**Remediation:** Require: "Because [character did X], [plot consequence]. This reveals [meaning]."

### ELA.10.R.1.3 - POINT OF VIEW & PERSPECTIVE

**Standard:** Analyze how POV creates effects (suspense, humor, etc.)

**Required Literacy Skills:**

| Literacy Skill | Why It's Needed | How to Teach It (Days 1-6) |
|----------------|-----------------|---------------------------|
| **1. Vocabulary** | Must know "POV," "1st/3rd person," "limited/omniscient," "narrator," "perspective," "reliable/unreliable" | Day 1: Teach POV types with examples from familiar texts |
| **2. Comprehension** | Must track what narrator knows vs doesn't know | Day 1-2: Annotation of what narrator reveals/hides |
| **3. Close Reading** | Must find evidence of POV effects (suspense from limited knowledge, irony from omniscience) | Day 2: Mark moments where POV creates effect (reader knows more than character = suspense) |
| **4. Analytical Writing** | Must explain how POV choice creates specific effect | Days 3-4: Organizer - POV indicator → Effect created → Why author chose this POV |
| **5. Test-Taking** | MC: Identify POV and effect; CR: Analyze effectiveness of POV choice | Day 5: Model STOP for POV questions; CER for "What if told from different POV?" |
| **6. Metacognition** | Must ask: "How would this change if told from another character's POV?" | Days 3-4: Reflection - "What would we gain/lose with different POV?" |

**Common Student Gap:** Identifying POV but not analyzing effect

**Remediation:** Ask: "What does this narrator know/not know? How does that create [suspense/humor/irony]?"

## TEACHING LITERACY SKILLS THROUGH BENCHMARKS

### How to Integrate Skills into Benchmark Instruction

**DON'T:** Teach literacy skills in isolation, then teach benchmark separately
**DO:** Teach literacy skills AS TOOLS to master the benchmark

**Example (Theme Unit):**

❌ **Wrong Approach:**
- Day 1: Vocab lesson on random words
- Day 2: Teach theme (separate from other skills)
- Day 3: Writing practice (unconnected to theme)

✅ **Right Approach:**
- Day 1: Teach **theme vocabulary** (universal, symbolism) → Immediately apply to **comprehension** of neutral text → Students **annotate** theme evidence
- Day 2: **Close reading** of unit text to find theme → **Comprehension strategy** (questioning: "What message about X?") → **Annotate** theme development
- Days 3-4: **Analytical writing** via theme organizer → **Metacognitive** self-check ("Is my theme universal?") → Practice **test-taking** with theme MC
- Day 5: **Test-taking** focus (STOP for theme MC, CER for theme analysis)
- Day 6: Apply all skills in **assessment**

**Key:** Every literacy skill connects to theme; theme is the vehicle for practicing all literacy skills

## LITERACY SKILL SCAFFOLDS BY BENCHMARK

### Vocabulary Development Scaffolds

**For Theme:**
- Teach: theme, universal, development, convey, symbolism, motif
- Sentence frame: "The universal theme is _____ because _____"

**For Rhetoric:**
- Teach: repetition, parallelism, anaphora, rhetorical question, ethos, pathos, logos
- Anchor chart: Device name + definition + visual example

**For Argument:**
- Teach: claim, evidence, reasoning, counterclaim, rebuttal, bias
- C.E.R. poster visible all unit

### Close Reading Scaffolds

**For Theme:**
- Annotation code: 🎯 = theme evidence, ⭐ = symbol, 🔄 = character change

**For Rhetoric:**
- Annotation code: R = repetition, P = parallelism, RQ = rhetorical question, E = ethos, P = pathos, L = logos

**For Argument:**
- Annotation code: C = claim, E = evidence, R = reasoning, CC = counterclaim

### Analytical Writing Scaffolds

**For All Benchmarks (Days 3-4):**
- I Do row: 100% complete (model)
- We Do row: 50% complete (sentence starters)
- You Do w/ Partner: 25% complete (column headers only)
- You Do: Blank (assessment conditions)

**Remove scaffolds by Day 5** (prepare for independence)

## ASSESSMENT: How Literacy Skills & Benchmarks Connect

### Day 6 Mini-Assessment Structure

**MC Questions** (test literacy skills IN SERVICE of benchmark):
- 3-4 vocabulary questions (Skill #1) - test benchmark vocabulary
- 3-4 comprehension questions (Skill #2) - test understanding of text
- 2-3 close reading questions (Skill #3) - test evidence location
- 2-3 benchmark application questions (Skills #2-4) - test benchmark mastery

**CR Questions** (test integrated skills):
- Prompt uses benchmark language
- Students apply Skills #3 (find evidence), #4 (write analysis), #5 (use CER)
- Graded on: Evidence (close reading) + Reasoning (analytical writing) + Structure (test-taking)

**Example CR Prompt (Theme):**
```
Analyze how the author develops the theme of [theme] throughout the text. Use specific evidence from the beginning, middle, and end.

Required Skills:
- Vocabulary: Understand "theme," "develop," "throughout"
- Close Reading: Find evidence from 3 parts of text
- Analytical Writing: Explain how evidence shows theme development
- Test-Taking: Use CER structure
```

## TROUBLESHOOTING: When Literacy Skills Are Missing

### If Students Struggle with Benchmark, Diagnose Which Literacy Skill Is Missing

**Theme Example:**

**Problem:** Students can't analyze theme
**Diagnose:**
- Vocabulary issue? → Can they define "theme" vs "topic"?
- Comprehension issue? → Do they understand the plot?
- Close reading issue? → Can they find evidence of theme?
- Writing issue? → Can they explain how evidence shows theme?
- Metacognition issue? → Do they know what they don't understand?

**Fix:** Target the specific literacy skill that's missing (don't just reteach theme)

## INTEGRATION WITH UNIT BUILDING

### Phase 1: Unit Planning
- Identify target benchmark
- List required literacy skills for that benchmark (use matrix above)
- Plan which skills to teach which days

### Phase 2: Lesson Planning
- Day 1: Vocabulary + Comprehension (foundation)
- Day 2: Close Reading (evidence gathering)
- Days 3-4: Analytical Writing (benchmark organizer)
- Day 5: Test-Taking (strategies)
- Day 6: Integration (assessment)

### Phase 3: Materials Building
- Vocab list includes benchmark-specific words
- Organizer aligns to benchmark + requires close reading + writing
- Assessment tests literacy skills in service of benchmark
