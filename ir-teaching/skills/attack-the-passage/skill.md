# Attack the Passage Protocol — 10th Grade Intensive Reading

## Purpose

Attack the Passage is the **master test-taking strategy** for Intensive Reading. It transforms students from passive readers into active, strategic "hunters" of information through a 4-phase sequential protocol. Every phase builds on the last, moving students from pre-reading analysis to post-reading proof.

**The core philosophy:** Demystify the reading process. Make every step visible and repeatable. Students learn to think about their own thinking (metacognition), turning implicit cognitive tasks into explicit, manageable strategies.

**The core habit:** "Don't read blindly. Hunt with a plan."

---

## The Four Phases

```
PHASE 1: ATTACK THE QUESTIONS (Before you read)
    → Know what you're hunting for
         ↓
PHASE 2: ATTACK THE PASSAGE (While you read)
    → Become an evidence detective
         ↓
PHASE 3: ATTACK THE ANSWER CHOICES (After you read)
    → Eliminate wrong answers using STOP
         ↓
PHASE 4: ATTACK THE JUSTIFICATION (Prove your answer)
    → Quick justification during test, extended CER/RACE during review
```

**Key insight:** Phase 1 happens BEFORE reading. This is the single biggest shift — students analyze questions first, then read with purpose. This converts reading from passive intake to active, goal-oriented search.

## Phase 1: Attack the Questions

### The C-U-B Method for Questions

Students deconstruct every question BEFORE reading the passage using Circle-Underline-Box.

| Mark | What to Mark | Why |
|------|-------------|-----|
| **CIRCLE** | Key terms that tell you WHAT to find | Focuses attention on the cognitive task (infer, symbolize, develop, reinforce) |
| **UNDERLINE** | What the question is actually asking | Isolates the core task, prevents misinterpretation |
| **BOX** | Paragraph numbers or location clues | Maps where to look before reading even begins |

### Identify the Hunt

Once students circle the key term, they identify what type of "prey" they're hunting:

| If You Circled... | You're Hunting For... |
|-------------------|----------------------|
| **infer** | Hidden meaning — not directly stated |
| **develop** | How something changes or is built over time |
| **symbolize** | Deeper meaning beyond the literal object |
| **reinforce** | How two elements connect or support each other |
| **contrast** | Differences between two things |
| **central idea / theme** | The main message or lesson |
| **point of view** | Who's telling the story and how it affects meaning |
| **text structure** | How the text is organized (cause/effect, sequence, compare/contrast) |
| **figurative language** | Non-literal language (metaphor, simile, personification) and its effect |
| **mood / tone** | The feeling the text creates and the author's attitude |
| **ambiguity** | Multiple possible meanings or unresolved ideas |
| **layers of meaning** | Literal + figurative + symbolic meanings working together |

### Predict Before You Read

After C-U-B, students write a short prediction (3-5 words) next to each question:
- "Need character's actions"
- "Find a repeated object"
- "Look for dialogue"
- "Find figurative language in lines 1-4"

### Teacher Implementation (Phase 1)

**This is the most critical leverage point for the coach; mastery here is non-negotiable.**

**I Do script:**
"First, I'll underline the core question: 'How does the author use figurative language to create mood?' The key terms I'll circle are 'figurative language' and 'mood' — those tell me I'm hunting for non-literal language AND the feeling it creates. I see 'lines 5-7' so I'll box that — now I know exactly WHERE to look. My prediction: 'Find metaphor/simile in lines 5-7, name the mood.'"

## Phase 2: Attack the Passage

### The C-U-B Method for the Passage

The same C-U-B framework applies to the passage, but with evidence-gathering focus.

| Mark | What to Mark | Why |
|------|-------------|-----|
| **CIRCLE** | Key details, character actions, important moments, repeated words/ideas | Creates a visual library of potential evidence |
| **UNDERLINE** | Confusing words, unfamiliar vocabulary, complex sentences | Flags tricky spots to figure out |
| **BOX** | Paragraph numbers where you find important information | Acts as signposts for quick evidence retrieval |

### Margin Notes — The Secret Weapon

**Writing the question number in the margin is the single most critical connection between Phase 1 and Phase 2.** This transforms the passage into a personalized answer key.

| What You Notice | What to Write |
|----------------|---------------|
| Detail that answers a question | **Q#** (Q1, Q2, etc.) |
| Character trait or action | **Char:** + brief note |
| Theme or lesson clue | **Theme?** |
| Object that seems meaningful | **Symbol?** |
| Cause and effect | **C→E** |
| Conflict or turning point | **Conflict** or **TP** |
| Figurative language | **Fig lang:** + type |
| Mood shift | **Mood:** + word |
| Multiple meanings | **Ambig:** + note |

### Evidence Gathering Checklist

Before moving to Phase 3, students verify:
- [ ] Did I mark evidence for each question?
- [ ] Did I write Q# in the margins next to useful details?
- [ ] Can I point to WHERE in the text my answer comes from?

### Benchmark-Specific Phase 2 Adaptations

When annotating, students should focus their C-U-B based on the benchmark:

| Benchmark | Circle Focus | Underline Focus | Box Focus |
|-----------|-------------|-----------------|-----------|
| R.1.1 Literary Elements | Character names, setting details, plot events | How elements CONTRIBUTE to meaning | Dialogue, descriptions revealing character |
| R.1.2 Theme | Repeated ideas, symbols, realizations | Passages revealing the MESSAGE (not topic) | Evidence supporting theme statement |
| R.1.3 Point of View | Moments of growth, conflicting perspectives | Transformation or opposing viewpoints | Before/after evidence showing change |
| R.1.4 Poetry | Figurative language, imagery, ambiguous phrases | Lines with MULTIPLE possible meanings | Contrasts, paradoxes, shifts |
| R.2.1 Text Structure | Signal words (because, however, similarly) | Author's purpose/central idea | Headings, bold terms, features |
| R.2.2 Central Idea | Repeated concepts, key terms | Central idea statement (explicit or implicit) | Supporting details, examples |
| R.2.3 Purpose | Rhetorical devices, appeals | Author's purpose and choices | Specific authorial choices with para # |
| R.2.4 Argument | Claims, counterclaims, evidence types | Each author's main claim | Shared evidence both authors reference |
| R.3.1 Figurative Language | ALL figurative language | Mood the passage creates | Most powerful examples with para # |
| R.3.4 Rhetoric | Rhetorical devices + appeals | Author's purpose and audience | Specific rhetoric examples with para # |

## Phase 3: Attack the Answer Choices

### Check Text Features First

Before eliminating, scan for:
- **Titles / Headings** — What do they reveal about topic or structure?
- **Paragraph numbers** — Do answer choices point to specific locations?
- **Formatting** — Sections, bullet points, italics signaling importance?
- **Footnotes** — Key terms defined or context provided?

### The STOP Strategy

**For the complete STOP teaching guide, anchor charts, ESOL modifications, and practice templates, see `stop-strategy`.**

| Letter | Type | What It Means | What to Do |
|--------|------|--------------|------------|
| **S** | **Silly** | Obviously wrong, not in text at all | Cross out immediately |
| **T** | **Tricky** | Partially true OR sounds good but doesn't answer THIS question | Cross out — don't fall for it |
| **O** | **Opposite** | Says the reverse of what the text actually says | Cross out — it's a trap |
| **P** | **Proven** | Directly supported by text evidence you can point to | THIS IS YOUR ANSWER |

### STOP in Action — Decision Tree

For each answer choice, ask sequentially:

1. **Is this Silly?** Can I find this ANYWHERE in the text? No → eliminate
2. **Is this Tricky?** Is part of it true but part wrong? Does it sound good but not answer THIS question? Yes → eliminate
3. **Is this Opposite?** Does the text say the reverse of this? Yes → eliminate
4. **Is this Proven?** Can I point to specific evidence? Yes → select it

### When Two Choices Remain

If stuck between two answers:
- Which one has STRONGER text evidence?
- Which one answers the EXACT question asked?
- Re-read the question stem — did you miss a key word?

## Phase 4: Attack the Justification

### Quick Justification (During Cold Read / Test)

One-sentence proof written immediately after selecting an answer. Serves as rapid self-monitoring.

**Frame:** "The answer is ___ because [brief reason connected to text]."

**Examples:**
- "The answer is B because the character's dialogue shows she gives warnings."
- "The answer is C because the object is mentioned repeatedly at key moments."
- "The answer is A because paragraph 3 directly states the cause."

### Extended Justification (During Review)

Uses the appropriate framework based on text type:

#### CER for Literary Texts (Fiction, Poetry, Drama)

| Component | What to Write |
|-----------|--------------|
| **CLAIM** | The correct answer is ___ because [what it means]. |
| **EVIDENCE** | In paragraph ___, the text states "____________." |
| **REASONING** | This proves the answer because [explain the connection to the question]. |

**Why CER for literature:** Literary analysis requires interpretive Claims defended with Evidence and interpretive Reasoning. The focus is on building a compelling argument for meaning.

**CER Sentence Frames:**
- Claim: "The correct answer is ___ because it shows that _____."
- Evidence: "In paragraph ___, the text states, '_____.' "
- Reasoning: "This proves the answer because _____."

#### RACE for Informational Texts (Nonfiction, Speeches, Arguments)

| Component | What to Write |
|-----------|--------------|
| **RESTATE** | Turn the question into a statement. |
| **ANSWER** | Provide a direct, specific answer. |
| **CITE** | Quote text evidence with paragraph number. |
| **EXPLAIN** | Explain how evidence proves the answer. |

**Why RACE for informational:** Informational questions often have more explicit, fact-based answers. RACE's structure begins by Restating the question (keeping focus), then provides a direct Answer followed by Evidence and Explanation.

**Note:** For units that already use RACE as their default CR framework (most IR units), use RACE for both literary and informational extended justifications. CER is an alternative when argument-style justification is more appropriate. **One framework per unit — never mixed.** See `race-strategy` and `cer-writing-guide` for full details.

### Error Analysis

When an answer is wrong, students diagnose their error:

| Code | Meaning |
|------|---------|
| **S** | Fell for Silly answer |
| **T** | Fell for Tricky answer |
| **O** | Chose Opposite of text |
| **MR** | Misread question |
| **R** | Rushed |
| **V** | Unfamiliar vocabulary |
| **NE** | Didn't find/use evidence |

**Error analysis is critical for data collection and driving instruction.** After assessment, students use the Error Type Key on their Attack the Passage Tracker. Teachers can quickly see if a student consistently falls for "Tricky" answers or "Misreads" questions, allowing targeted re-teaching.

## 4-Day Implementation Cycle

The protocol maps to a 4-day cycle per text. See `test-prep-unit` for the full 16-day unit framework.

| Day | Phase Focus | Teacher-Led Center | Independent Center |
|-----|-----------|-------------------|-------------------|
| **Day 1** | Phase 1 + 2 | Cold read + 10 MC + quick justifications | Question preview (C-U-B questions ONLY, no passage yet) |
| **Day 2** | Phase 1 Deep Dive | Model question stem analysis, identify benchmarks | Question stem worksheet, hunt predictions |
| **Day 3** | Phase 3 + 4 (Quick) | Model STOP elimination + CER/RACE justification | Write CER/RACE justifications, complete tracker |
| **Day 4** | Phase 4 (Extended) + Review | Review game + guide extended CER/RACE | Draft extended response, complete tracker, self-assessment |

**Coaching Rationale:** The cycle is deliberately sequenced — Day 1 gives a holistic reading experience, Day 2 narrows focus to question analysis, Day 3 teaches systematic elimination, Day 4 consolidates with review and extended writing. Each day builds on the last.

## Review Games (Day 4)

### 4 Corners (15-20 min)
- Label corners A, B, C, D
- Display question, students move to their answer's corner
- Each corner defends with text evidence
- Incorrect corners identify their error type (S/T/O)
- **Why:** Physical movement, public commitment, peer discussion

### Defense Court (15-20 min)
- Pairs: one "prosecutor," one "defendant"
- Prosecutor: "Why did you choose ___ for Question ___?"
- Defendant justifies using CER/RACE format
- Prosecutor challenges weak evidence
- Switch roles per question
- **Why:** Oral rehearsal of justification, peer accountability

### Beat the Teacher (10-15 min)
- Teacher deliberately selects a WRONG answer and justifies it
- Students identify: What's wrong with this reasoning?
- First student to correctly identify the flaw earns a point
- Discuss what error type the teacher "fell for"
- **Why:** Models metacognition, makes errors visible and learnable

### Question Stem Jeopardy (20-25 min)
- Categories: Literary Elements, Theme, Central Idea, Inference, Evidence
- Teams select category + point value (100-500)
- Must answer AND provide text evidence
- Correct answer + evidence = full points; answer only = half
- **Why:** Competitive, reinforces evidence requirement

### Error Auction (15 min)
- Display common wrong answers from class (anonymized)
- Teams bid points to "buy" the error
- Winning team explains: What error type? Why wrong?
- Correct = earn points; wrong = lose points
- **Why:** Analyzes errors without embarrassment, gamifies metacognition

## ESOL Modifications

### By Proficiency Level

**ESOL Levels 1-2 (Entering/Beginning):**
- Phase 1: Start with C-U only (Circle and Underline). Add Box once comfortable.
- Phase 2: Allow bilingual margin notes. Provide vocabulary pre-teach before cold read.
- Phase 3: Start with S and P only (find Silly, find Proven). Add T and O after 2-3 texts.
- Phase 4: Quick justification can be 1 phrase instead of 1 sentence. Allow oral justification to partner before writing.
- Provide chunked text with check-ins during cold read.
- Partner with strong reader for Days 2-3 discussion.

**ESOL Level 3 (Developing):**
- Full C-U-B with sentence frames for justification.
- Extended time for cold read at teacher center if needed.
- Word bank for mood, tone, and analytical vocabulary.

**ESOL Levels 4-5 (Expanding/Bridging):**
- Full protocol with standard expectations.
- May need vocabulary support for question stems only.

### ESOL Sentence Frames for Each Phase

```
Phase 1: "This question is asking me to find ___."
Phase 2: "In paragraph ___, I notice ___ which connects to Q___."
Phase 3: "This answer is [S/T/O/P] because ___."
Phase 4 Quick: "The answer is ___ because ___."
Phase 4 Extended:
  Claim: "The correct answer is ___ because it shows ___."
  Evidence: "In paragraph ___, the text states '___.' "
  Reasoning: "This proves ___ because ___."
```
