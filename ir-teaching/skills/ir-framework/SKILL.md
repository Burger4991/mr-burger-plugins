# IR Framework - 6-Day Cycle

## Daily Structure
Every day follows this structure:

**1. Context Clues Bellringer (5-10 minutes)**
- Warm-up before rotations begin
- Uses vocabulary from the unit text
- Students infer word meanings from context
- 5 words per day
- Self-graded or quickly reviewed

**2. Three Rotations (20 minutes each):**
1. **Teacher-Led (TL)** - Direct instruction, modeling, guided practice
2. **Independent (IND)** - Student applies learning independently
3. **Technology (Tech)** - Read 180 differentiated practice

**Total Daily Time:** 5-10 min bellringer + 60 min rotations = 65-70 minutes

## 6-Day Cycle Breakdown

### Days 1-2: Build Foundation

**Bellringer:**
- 5 vocabulary words from unit text
- Students use context clues to infer meanings
- Quick review (2-3 minutes) or self-check
- Builds vocabulary before reading

**Teacher-Led:**
- Teach vocabulary/concepts/strategies
- Use neutral texts (not the unit text yet)
- Pre-teach essential skills
- Review bellringer words if needed

**Independent:**
- First read of unit text with margin questions
- Convert margin questions to multiple choice (exact wording)
- Add Planning Card/9-10 Guide stems for benchmark practice

**Technology:**
- Set Read 180 targets (minutes/segments/zones)
- Student tracking logs
- Monitor engagement data

### Days 3-4: Analyze Text

**Bellringer:**
- Continue context clues practice (5 more words each day)
- Vocabulary from unit text
- Reinforces words students will encounter

**Teacher-Led:**
- Analyze unit text via benchmark organizers
- **Gradual Release**: I Do → We Do → You Do w/ Partner → You Do
- Embed mini-lessons for benchmark skills
- Model thinking process explicitly

**Independent:**
- Complete organizer rows (You-Do)
- Vocabulary practice activities
- Apply strategies from TL independently

**Technology:**
- Continue Read 180 progression
- Use data for Day 5 small-group pulls
- Track student zones and achievements

### Days 5-6: Assess & Apply

**Bellringer:**
- Final vocabulary practice (5 more words each day)
- Total: 30 words across 6 days from unit text
- Optional: Review previous words

**Teacher-Led:**
- Model STOP protocol for multiple choice
- Model CER writing process
- Conduct mini-assessments
- Pull small groups based on Read 180 data

**Independent:**
- Mini-assessment completion
- Justification sheets (Proven + Why Not)
- Independent CER responses

**Technology:**
- Final Read 180 segment
- Review student progress reports
- Celebrate zone advancements

## Gradual Release Model (Critical for Days 3-4)

**I Do (Teacher Models):**
- Think aloud while completing first row of organizer
- Show exactly what you're thinking
- Demonstrate text evidence location

**We Do (Guided Practice):**
- Complete next row together
- Call on students for input
- Correct misconceptions immediately

**You Do with a Partner:**
- Students work in pairs on next section
- Teacher circulates and monitors
- Provide feedback in real-time

**You Do (Independent):**
- Students complete remaining rows alone
- Teacher checks for understanding via exit ticket
- Collect data for next day's grouping

## Scaffolding Across Days
- **Day 1-2**: Maximum support (sentence frames, word banks, exemplars)
- **Day 3-4**: Partial support (sentence starters, reference materials)
- **Day 5-6**: Minimal support (independence expected, intervention available)

## Read 180 Integration
- Set weekly targets: minutes on task, segments passed, zone goals
- Students log progress daily
- Use reports to identify struggling students
- Pull small groups Day 5-6 based on data

## Teacher Monitoring Strategies
- **Active scanning**: Check screens during Tech time
- **Real-time feedback**: Verbal corrections during rotations
- **Check for understanding**: Exit tickets, cold calls, quick writes
- **Performance tracking**: Read 180 reports, organizer completion, assessment scores
- **Formative feedback system**: See `feedback-system` skill for complete implementation
  - Days 1-2: Spot-check 5-8 students during IND rotation (2-3 min each)
  - Days 3-4: Collect organizers end of Day 4, return with feedback Day 5
  - Days 5-6: Return assessments with feedback forms, students complete Action Plan

## Detailed Teacher-Led Center Expectations

### Days 1-2: Foundation Building
**Teacher Actions:**
- Introduce and explain benchmark(s) - explicitly state what's being addressed
- Review and clarify benchmark language with students
- Set clear learning goals (what students will learn over 3-5 days)
- Teach the literary concept with clear, direct instruction
- Model the analytical process - demonstrate how to analyze texts with purpose
- Show examples and think aloud to make the process visible
- Use neutral texts (NOT the unit text yet)

**Key Focus:** Vocabulary, concepts, strategies

### Days 3-4: Text Analysis (Gradual Release Days)
**Teacher Actions:**
- Guide the analytical reading process step-by-step
- Chart examples and think aloud to make process visible
- Structure learning for clarity (transparent, easy to follow)
- Embed mini-lessons for benchmark skills
- Check for understanding frequently, provide timely feedback

**Key Focus:** Benchmark organizer completion using I Do → We Do → You Do w/ Partner → You Do

**Mini-Lesson Examples:**
- "How to identify universal themes" (not just themes)
- "How to track theme development from beginning to end"
- "How to cite text evidence with paragraph numbers"
- "How to eliminate incorrect answer choices using STOP"

### Days 5-6: Assessment Preparation
**Teacher Actions:**
- Provide explicit directions for completing assessments
- Model STOP protocol (Silly, Tricky, Opposite, Proven)
- Model CER writing (Claim, Evidence, Reasoning)
- Model RACE writing (Restate, Answer, Cite, Explain)
- Continue support and mini-lessons as needed
- Pull small groups based on Read 180 data

**Key Focus:** Assessment readiness, intervention, confidence-building

## Detailed Independent Center Expectations

### Days 1-2: First Read
**Student Actions:**
- Read the entire text independently
- Answer comprehension questions (margin questions converted to MC)
- Annotate evidence to support conclusions
- Use process of elimination to remove two most incorrect responses

**Materials Needed:**
- Student packet with full text and paragraph numbers
- Margin questions converted to MC using exact wording from source
- Planning Card/9-10 Guide stems for benchmark practice

**Teacher Feedback Actions (Day 2):**
- Circulate with IR-Feedback-Form on clipboard
- Spot-check 5-8 students (2-3 min each)
- Quick verbal feedback on bellringers and comprehension
- Students transcribe feedback in Verbal Feedback Notes section

**Optional:** Tier 3 students can do peer reading instead of independent

### Days 3-4: Application & Practice
**Student Actions:**
- Complete vocabulary activity
- Complete "You Do" components of benchmark-based graphic organizers
- Complete other analytical activities from Teacher-Led
- Make corrections to previous work based on Days 1-2 feedback

**Teacher Feedback Actions (End of Day 4):**
- Collect completed organizers
- Review 8-10 organizers during planning period (3-4 min each)
- Attach IR-Feedback-Form with checked boxes + written feedback
- Return Day 5 with 1-minute verbal feedback per student
- Students transcribe feedback

**Critical:** Activities must be self-running (students can complete without teacher help)

### Days 5-6: Independent Assessment
**Student Actions:**
- Independently complete mini-assessment (practice version, not graded yet)
- Complete answer justification sheet (STOP analysis: Proven + Why Not for each choice)
- Use RACE or CER format for constructed responses
- Complete Student Action Plan on feedback form before leaving (3 specific revisions)

**Teacher Feedback Actions (Day 6 or after):**
- Grade practice assessments with IR-Feedback-Form attached (3-4 min each)
- Give practice score (not recorded in gradebook yet)
- Write ONE specific "Add" or "Revise" action step
- 30-second verbal feedback per student
- Students complete Action Plan: 3 revisions they'll make at home
- Students submit revised work online with feedback form attached (THIS gets graded)

## Detailed Technology Center Expectations

### Read 180 Program Structure (Days 1-5)

**Teacher Actions:**
- Set targets: minutes/segments/zones
- Actively scan screens (monitoring software or visual checks)
- Provide real-time feedback verbally
- Check for understanding with performance reports
- Spot misconceptions early by reviewing responses
- Redirect off-task behavior quickly and discreetly
- Track engagement data (Lessons Passed, Time on Task)

**Student Activities:**
- Explore Anchor Videos (context building)
- Reading Practice (differentiated passages)
- Language - Vocabulary Practice (word learning)
- Fluency - Spelling and Reading Practice (pronunciation, speed)
- Writing - Crafting Essays (short writing tasks)
- Success Zone (review and reinforcement)

**Data Use:**
- Day 5: Use Read 180 data for small-group pulls
- Identify struggling students for intervention
- Celebrate zone advancements to build motivation

**Student Tracking:**
- Students log minutes, segments passed, zones completed daily
- Teacher reviews weekly progress reports
- Set weekly goals for minutes on task
