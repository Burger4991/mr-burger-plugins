# Teaching Templates for 10th Grade IR

## File Naming Convention
Format: `[UnitName]_[Days]_[Type]_YYYYMMDD.[extension]`

Examples:
- `StoryOfAnHour_D1-6_LessonPlan_20251105.md`
- `StoryOfAnHour_D1-6_StudentPacket_20251105.md`
- `StoryOfAnHour_D1-6_Slides_20251105.pptx`
- `StoryOfAnHour_D1-6_AnswerKey_20251105.md`

**IMPORTANT:** All deliverables must use proper file extensions:
- Teacher Lesson Plans: `.md` (Markdown format)
- Student Packets: `.md` (Markdown format)
- Slide Decks: `.pptx` (NOT .md)
- Answer Keys: `.md` (Markdown format)


## 6-Day Unit Structure

Each unit follows this pattern:
- **Days 1-2**: Build foundation with vocabulary/concepts using neutral texts
- **Days 3-4**: Analyze unit text via benchmark organizers (gradual release)
- **Days 5-6**: Model STOP for MC, CER writing, mini-assessments

**Practice → Feedback → Revise → Submit Workflow:**
- **In class (Days 1-6):** Students complete practice work, receive formative feedback using IR-Feedback-Form
- **At home:** Students revise based on feedback
- **Online submission:** Students submit revised work with feedback form attached for grading
- See `feedback-system` skill for complete implementation details

## Daily Structure (Each Day)

**EVERY day includes (in this order):**

1. **Context Clues Bellringer** (5-10 minutes)
   - Uses vocabulary from unit (2 words per day: 1 content + 1 academic)
   - Students infer meaning from context before instruction
   - 12 words total per 6-day unit (6 content + 6 benchmark academic)
   - Self-graded or quickly reviewed

2. **Rotation Pattern** (20 minutes each):
   - Teacher-Led (TL) - Direct instruction
   - Independent (IND) - Self-running tasks
   - Technology (Tech) - Read 180

**Total daily time:** 5-10 min bellringer + 60 min rotations = 65-70 minutes

## Core Deliverables (In Order)

**Three synchronized files** (must match across all three):
1. **Teacher Lesson Plan (.md)** - Teacher-facing, includes bellringer answer keys
2. **Student Packet (.md)** - Student-facing, day-by-day format with bellringers
3. **Slide Deck (.pptx)** - For projection during Teacher-Led instruction

**Additional files:**
4. Answer Key & Exemplars (.md) - Separate from student packet
5. **Feedback Forms (.md)** - IR-Feedback-Form for practice→feedback→revise workflow (see `feedback-system` skill)
6. Exit Tickets (.md) - Optional
7. Cover Pages (.md) - Optional

## Key Protocols to Embed
- **STOP**: Silly, Tricky, Opposite, Proven (for MC)
- **CER**: Claim, Evidence, Reasoning
- **RACE**: Restate, Answer, Cite, Explain

## Differentiation Requirements (Daily)
- 2+ ESOL strategies (rotated)
- 1+ visual organizer
- 1+ modification for ESE/ESOL students
- Groups: high/med/low distributed by ability, not label

## Detailed Deliverable Specifications

### 1. Teacher Lesson Plan (`[Unit]_D1-6_LessonPlan_YYYYMMDD.md`)

**FORMAT:** Markdown (.md)

**STRUCTURE:** Day-by-day breakdown (6 days total)

**Each Day Must Include:**

#### Bellringer Section (5-10 min)
- **Answer Key** with acceptable variations
- Context clues that students should identify
- How to review (whole class, self-check, partner check)
- Tier 2/3 support notes

#### Teacher-Led Center (20 min)
- Learning objective (specific, measurable)
- Materials needed (with packet page numbers)
- Procedure with timing:
  - I DO (teacher models)
  - WE DO (guided practice)
  - YOU DO w/ PARTNER (structured practice)
  - YOU DO (independent preview)
- ESOL strategies embedded (labeled, minimum 2)
- Engagement strategies (TWPS, Cold-Call, etc.)
- Formative assessment method
- Teacher script/talking points for modeling

#### Independent Center (20 min)
- What students do (step-by-step)
- Materials needed (packet page numbers)
- Self-running verification (can they do this alone?)
- How to check for understanding
- Intervention notes for struggling students

#### Technology Center (20 min)
- Read 180 targets (minutes, segments, zones)
- Monitoring strategy
- Data collection notes
- Small group pull criteria (if applicable)

#### Assessment/Reflection
- What to collect from students
- What to check for understanding
- Data to track for next day planning
- Adjustments needed

**Must Also Include:**
- "Pulled from:" citations (IR Framework, Planning Cards, BEST Standards, Benchmark Cards)
- Benchmark number and exact clarification language
- Exemplar model rows for organizers (what proficient looks like)
- Differentiation summary
- **Feedback timing notes** (see `feedback-system` skill):
  - Days 1-2: "Spot-check 5-8 students during IND rotation (2-3 min each)"
  - Days 3-4: "Collect organizers end of Day 4, return with feedback Day 5"
  - Days 5-6: "Return assessments with feedback forms, students complete Action Plan before leaving"

**Tone:** Professional, teacher-facing

**Format:**
- Font: Calibri, 11-12pt
- Margins: 1 inch
- Headers: Unit name, Day number, Benchmark
- Page numbers
- Clear section breaks between days

### 2. Student Packet (`[Unit]_D1-6_StudentPacket_YYYYMMDD.md`)

**FORMAT:** Microsoft Word (.md)

**STRUCTURE:** Day-by-day (NOT activity-by-activity)

**Cover Page Must Include:**
- Unit title
- Benchmark focus (student-friendly language)
- Student name field, Date, Period
- Days 1-6 label
- **Practice work note:** "This is PRACTICE work. You will receive feedback in class and revise before submitting online for a grade."
- Table of contents (optional but recommended)

**EACH DAY PAGE STRUCTURE:**

```
Name: _________________ Date: ________ Period: ___

===================================
DAY [#]: [Title/Focus]
===================================

📚 CONTEXT CLUES BELLRINGER (5-10 minutes)
[Bellringer content - see format below]

---

👨‍🏫 TEACHER-LED CENTER (20 minutes)
[Organizers, guided practice, notes sections]
[Students work on this DURING Teacher-Led instruction]

---

📝 INDEPENDENT CENTER (20 minutes)
[Self-running activities]
[Students complete this ALONE without teacher help]

```

**Context Clues Bellringer Format:**

```
📚 CONTEXT CLUES BELLRINGER

Directions: Read each sentence from "[Text Title]." Use context clues to determine the meaning of the underlined word. Write your definition.

1. "[Full sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________

2. "[Full sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________

[2 words total: 1 content area word from text + 1 benchmark academic word]
```

**Vocabulary Types:**
- **Content Area Words (6 total across 6 days):** Words that appear IN the unit text. Students use context from the passage.
- **Benchmark Academic Words (6 total across 6 days):** Words related to literary analysis (e.g., characterization, mood, imagery, irony, interaction, figurative language).

**Each day = 2 words (1 content + 1 academic)**

**Day 1-2 Independent Section Typically Includes:**
- First read of unit text (with paragraph numbers)
- Comprehension questions (margin questions → MC, exact wording)
- Vocabulary practice activities
- Annotation tasks

**Day 3-4 Teacher-Led Section Typically Includes:**
- Benchmark organizer with gradual release rows:
  - I DO row (completed as example)
  - WE DO row (sentence starters/partial completion)
  - YOU DO w/ PARTNER row (structured prompts)
  - YOU DO row (blank for independent work)

**Day 3-4 Independent Section Typically Includes:**
- Complete organizer rows marked "YOU DO"
- Practice applying skills from Teacher-Led
- Extension activities

**Day 5-6 Independent Section Typically Includes:**
- Mini-assessment (MC with STOP analysis)
- Constructed response (CER/RACE format)
- Answer justification sheets

**Must Include Throughout:**
- Student-friendly directions (numbered, bold key words)
- Self-running tasks (can complete without teacher)
- Sentence frames for ESOL support
- Word banks where appropriate
- "Need Help?" reminders (reread → check notes → ask partner → raise hand)
- Plenty of white space, large font (12pt minimum)
- Icons for visual clarity (📚📝👨‍🏫💻)

**Tone:** Friendly, encouraging, student-facing

**Format:**
- Font: Calibri, 12pt minimum
- Margins: 1 inch
- Double-space between sections
- Icons for visual appeal
- Page numbers
- Clear day breaks (page break between days recommended)

### 3. Slide Deck (`[Unit]_D1-6_Slides_YYYYMMDD.pptx`)

**FORMAT:** Microsoft PowerPoint (.pptx)

**STRUCTURE:** Organized by day (slides grouped by day)

**Each Day Slide Set Includes:**

#### Slide 1: Day [#] Agenda
- Learning goal for the day
- Activities overview
- Benchmarks addressed (if applicable)

#### Slide 2: Bellringer Slide
- Display bellringer for students to see
- Can be projected while students work
- Shows same sentences as student packet

#### Slides 3-X: Teacher-Led Instruction
- Benchmark introduction (Days 1-2 or as needed)
- Vocabulary slides with visuals (if teaching new words)
- I DO modeling slides (completed examples)
- WE DO guided practice slides (prompts for class discussion)
- YOU DO preview (what students will do independently)

**Vocabulary Slides Must Include:**
- Word, definition, visual/icon
- Example sentence from unit text
- Gesture or memory aid

**I DO Slides Must Include:**
- Complete model response
- Teacher think-aloud prompts (in notes section)
- Step-by-step process visible

**WE DO Slides Must Include:**
- Guiding questions for class
- Fill-in-the-blank for students to complete together
- Sentence frames

**Must Include Throughout:**
- Minimal text per slide (bullets, not paragraphs)
- Large font (24pt+ for body, 36pt+ for headings)
- High-contrast colors
- Images/icons on every slide for ESOL support
- Consistent color scheme throughout unit
- Bellringer slide for EVERY day

**Tone:** Visual, concise, student-facing

**Format:**
- PowerPoint 2016+ format
- Standard 16:9 slide ratio
- Professional theme/template
- Animations minimal (avoid distractions)
- Readable from back of classroom

### 4. Answer Key & Exemplars (`[Unit]_D1-6_AnswerKey_YYYYMMDD.md`)

**FORMAT:** Microsoft Word (.md)

**STRUCTURE:** Mirrors student packet day-by-day

**Each Day Must Include:**

#### Bellringer Answer Key
- Word: definition
- Context clues: [list clues from sentence]
- Accept variations: [acceptable student responses]

#### Comprehension Questions (if applicable)
- Correct answer with letter choice
- STOP justification:
  - Why correct answer is PROVEN (with paragraph evidence)
  - Why 1-2 wrong answers are S/T/O
- Text evidence cited

#### Organizer Exemplars
- Completed organizer showing proficient work
- All rows filled in (including I DO, WE DO, YOU DO)
- Text evidence with paragraph numbers
- What makes this proficient (notes for teacher)

#### Assessment Items
- MC answers with STOP analysis for each choice
- CER/RACE exemplar responses
- Rubric with point values
- Common errors to watch for

**Must Include:**
- Text evidence cited with paragraph numbers for EVERY answer
- Multiple acceptable variations (where applicable)
- Scoring rubric (points per section)
- Notes on what proficient/advanced looks like

**Tone:** Professional, teacher-facing

**Format:**
- Mirror student packet structure exactly
- Answers in bold or highlighted for easy scanning
- Include point values next to each item
- Section breaks between days

## Context Clues Bellringer Specifications (CRITICAL)

### Purpose
- Activate prior knowledge before lesson
- Pre-teach vocabulary from unit text (not random words)
- Build context clue skills
- Quick, low-stakes warm-up
- 5-10 minutes (does NOT cut into 20-minute rotations)

### Selection Criteria for Vocabulary Words
- **Source:** ONLY from the unit text (the story/article/poem students will analyze)
- **Difficulty:** Tier 2 academic or domain-specific words students may not know
- **Context-Rich:** Sentence must have strong context clues
- **Distribution:** Spread across the 6 days (5 words × 6 days = 30 words total)
- **Avoid:** Words students already know or words with no context support

### Format Requirements

**Student Packet Bellringer (each day):**
```
📚 CONTEXT CLUES BELLRINGER

Directions: Read each sentence from "[Unit Text Title]." Use context clues to determine the meaning of the underlined word. Write your definition.

1. "[Complete sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________________

2. "[Complete sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________________

3. "[Complete sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________________

4. "[Complete sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________________

5. "[Complete sentence from text with target word underlined]" (¶#)

   Word: __________     My definition: _________________________________
```

**Lesson Plan Bellringer Answer Key (each day):**
```
DAY [#] BELLRINGER ANSWER KEY

1. Word: [word]
   Definition: [precise definition]
   Context clues: [what in the sentence helps students figure it out]
   Accept variations: [similar definitions students might write]

2. Word: [word]
   Definition: [precise definition]
   Context clues: [what in the sentence helps]
   Accept variations: [acceptable alternatives]

[Continue for all 5 words]

HOW TO REVIEW:
- Students self-check or partner-check (2 min)
- Quickly review 1-2 words whole-class if needed (2 min)
- Tier 3 students: provide word bank with definitions
```

**PowerPoint Bellringer Slide (each day):**
```
[Slide displays all 5 sentences with underlined words]
[Students copy into packet or just write definitions]
[Can click to reveal answers after students work]
```

### Example Bellringer (Day 1, "Story of an Hour")

**In Student Packet:**
```
📚 CONTEXT CLUES BELLRINGER - DAY 1

Directions: Read each sentence from "The Story of an Hour." Use context clues to determine the meaning of the underlined word.

1. "Knowing that Mrs. Mallard was afflicted with a heart trouble, great care was taken to break to her as gently as possible the news of her husband's death." (¶1)

   Word: afflicted     My definition: _________________________________

2. "She was young, with a fair, calm face, whose lines bespoke repression and even a certain strength." (¶7)

   Word: repression     My definition: _________________________________

3. "There was something coming to her and she was waiting for it, fearfully. What was it? She did not know; it was too subtle and elusive to name." (¶8)

   Word: elusive     My definition: _________________________________

4. "Now her bosom rose and fell tumultuously." (¶9)

   Word: tumultuously     My definition: _________________________________

5. "She did not stop to ask if it were or were not a monstrous joy that held her." (¶10)

   Word: monstrous     My definition: _________________________________
```

**In Lesson Plan Answer Key:**
```
DAY 1 BELLRINGER ANSWER KEY

1. afflicted = suffering from, troubled by
   Context clues: "heart trouble," "great care was taken"
   Accept: sick with, has a problem with, suffering from, troubled by

2. repression = holding back feelings or desires, suppressing
   Context clues: "lines bespoke" (her face showed), contrasted with "strength" (she was strong but held back)
   Accept: holding back, suppressing, keeping feelings inside

3. elusive = difficult to grasp or define, hard to identify
   Context clues: "too subtle," "did not know," "could not name"
   Accept: hard to understand, difficult to grasp, unclear, hard to identify

4. tumultuously = wildly, with great agitation or turmoil
   Context clues: "bosom rose and fell" (breathing heavily)
   Accept: wildly, in a troubled way, with turmoil, agitatedly, excitedly

5. monstrous = shocking, unnatural, unnaturally large
   Context clues: "did not stop to ask if" (questioning if it was acceptable), paired with "joy" (unusual combination)
   Accept: shocking, wrong, unnatural, not normal

HOW TO REVIEW:
- Students self-check definitions (2 min)
- Ask: "Which word was hardest? Why?" (1 min)
- Clarify 1-2 words as needed (2 min)
- Tier 3: Provide word bank: "suffering from, holding back, hard to grasp, wildly, shocking"
```

## Three-File Synchronization Requirements

**CRITICAL:** The following 3 files must be synchronized:
1. Teacher Lesson Plan (.md)
2. Student Packet (.md)
3. Slide Deck (.pptx)

### What Must Match Across All Three Files:

✅ **Vocabulary**
- Same words taught on same days
- Same definitions
- Same example sentences
- Bellringer vocabulary matches lesson vocabulary

✅ **Organizer Content**
- I DO rows must be identical
- WE DO prompts must match
- Questions/prompts must be worded the same
- Sentence frames must match

✅ **Examples Used**
- Model examples in lesson plan = examples on slides
- Examples in student packet match lesson plan
- Consistent use of text evidence (same paragraph citations)

✅ **Day Structure**
- Same activities on same days across all files
- Timing must align
- Sequence of activities must match

✅ **Questions/Prompts**
- Assessment questions in student packet = questions referenced in lesson plan
- Discussion prompts on slides = prompts in lesson plan
- Exact wording for consistency

### How Sync-Coordinator Handles Edits:

**When user edits Teacher Lesson Plan:**
- Updates student packet to match
- Updates slides to reflect changes
- Reports what was synchronized

**When user edits Student Packet:**
- Updates lesson plan to match
- Updates slides to match
- Reports what was synchronized

**When user edits Slides:**
- Updates lesson plan to match
- Updates student packet to match
- Reports what was synchronized

## Writing Protocols to Embed

### STOP (Multiple Choice Analysis)
**Use:** Days 5-6, assessment preparation
- **S**illy: Obviously incorrect, easy to eliminate
- **T**ricky: Sounds partially correct but misses key concept
- **O**pposite: States reverse of correct answer
- **P**roven: Correct answer, can be proven with text evidence

Students must label each choice (S/T/O/P) and justify WHY for each.

### CER (Claim-Evidence-Reasoning)
**Use:** Argumentative and analytical writing
- **Claim:** Your position or interpretation (answers the prompt, debatable)
- **Evidence:** Specific facts, quotes, or data from text with paragraph citations
- **Reasoning:** Explains HOW evidence supports claim, makes thinking visible

### RACE (Restate-Answer-Cite-Explain)
**Use:** Short-answer constructed responses
- **R**estate: Turn question into a statement using key words
- **A**nswer: Provide clear, direct answer with academic vocabulary
- **C**ite: Use quotation marks, include paragraph number, choose most relevant evidence
- **E**xplain: Connect evidence back to answer - "This shows," "This demonstrates," "This reveals"
