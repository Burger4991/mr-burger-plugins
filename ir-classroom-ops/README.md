# ir-classroom-ops

Day-to-day classroom logistics for 10th grade Intensive Reading.

6 skills for behavior tracking, parent contact logging, observation prep, PD tracking, substitute folder building, and unit distribution workflows.

## Install

### Claude Code

```
/plugin marketplace add Burger4991/mr-burger-plugins
/plugin install ir-classroom-ops@mr-burger-plugins
```

### Cowork

1. Download `ir-classroom-ops.plugin` from [Releases](https://github.com/Burger4991/mr-burger-plugins/tree/main/packages)
2. Open Cowork settings → Plugins → Install from file

### Development (local symlinks)

```bash
cd mr-burger-plugins && ./scripts/setup.sh
```

## What's included

- **behavior-tracker** — Document in-class student behavior patterns
- **parent-contact-log** — Log parent/guardian communication events
- **observation-prep** — Generate lesson documentation for admin walkthroughs
- **pd-tracker** — Track professional development hours toward Florida recertification
- **sub-folder-builder** — Build permanent substitute teacher reference folders
- **unit-distribution** — "Ship it" workflow after quality gate passes (print, upload, archive, share)
